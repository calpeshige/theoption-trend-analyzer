# TheOption Trend Analyzer v2.0.0 - 使い方ガイド

## 🎯 概要

TheOptionのトレーディング画面でリアルタイムにトレンド分析を実行し、HIGH/LOWのシグナルを表示する Chrome 拡張機能です。

## ✨ 機能

### 📊 実装済み機能
- ✅ リアルタイム価格監視（1秒ごと）
- ✅ テクニカル指標計算
  - 移動平均線（MA5, MA20, MA50）
  - RSI（相対力指数）
  - ボリンジャーバンド
  - MACD
- ✅ トレンド総合判定（HIGH/LOW/NEUTRAL）
- ✅ 信頼度スコア表示
- ✅ 美しいUIパネル

## 📦 インストール方法

### 1. 拡張機能の読み込み

```bash
# Chromeを開く
chrome://extensions/

# 「デベロッパーモード」を ON にする
# 「パッケージ化されていない拡張機能を読み込む」をクリック
# このフォルダを選択: /Users/shige/Documents/拡張機能/theoption_trend/
```

### 2. 動作確認

1. TheOptionにアクセス: https://jp.theoption.com/trading
2. トレーディング画面が表示されたら、右上に「📊 トレンド分析」パネルが出現
3. コンソール（F12）で以下のログを確認:
   ```

   [TheOption Analyzer] 🔧 初期化開始

   ```

### シグナルの見方

#### ⬆ HIGH（緑色）
- **意味**: 価格が上昇する可能性が高い
- **推奨**: HIGHでエントリー
- **信頼度**: 60-90%

#### ⬇ LOW（赤色）
- **意味**: 価格が下降する可能性が高い
- **推奨**: LOWでエントリー
- **信頼度**: 60-90%

#### NEUTRAL（グレー）
- **意味**: トレンドが不明瞭
- **推奨**: エントリー見送り
- **信頼度**: 40%以下

## 📈 分析ロジック

### 判定基準

スコアリングシステムで総合判定:

| 指標 | HIGH方向 | LOW方向 |
|------|---------|---------|
| MA5 > MA20 | +2点 | -2点 |
| MA20 > MA50 | +1点 | -1点 |
| RSI < 30 (売られすぎ) | +3点 | - |
| RSI > 70 (買われすぎ) | - | -3点 |
| BB下限突破 | +2点 | - |
| BB上限突破 | - | -2点 |
| MACD上昇 | +1点 | -1点 |

**総合判定**:
- スコア ≥ 4点: 強いHIGHシグナル（信頼度70-90%）
- スコア ≥ 2点: HIGHシグナル（信頼度60-70%）
- スコア ≤ -4点: 強いLOWシグナル（信頼度70-90%）
- スコア ≤ -2点: LOWシグナル（信頼度60-70%）
- その他: NEUTRAL（信頼度40%）

### テクニカル指標の意味

#### MA（移動平均線）
- **MA5**: 直近5ポイントの平均価格
- **MA20**: 直近20ポイントの平均価格
- **MA50**: 直近50ポイントの平均価格
- **ゴールデンクロス**: MA5 > MA20 → 上昇トレンド
- **デッドクロス**: MA5 < MA20 → 下降トレンド

#### RSI（相対力指数）
- **0-30**: 売られすぎ → 反発の可能性（HIGH）
- **30-50**: 中立-弱気
- **50-70**: 中立-強気
- **70-100**: 買われすぎ → 反落の可能性（LOW）

#### ボリンジャーバンド
- **BB上限突破**: 買われすぎ → 反落警戒
- **BB下限突破**: 売られすぎ → 反発期待
- **バンド内**: 通常範囲

#### MACD
- **ヒストグラム > 0**: 上昇モメンタム
- **ヒストグラム < 0**: 下降モメンタム

## 🔧 トラブルシューティング

### パネルが表示されない

**原因**: 拡張機能が正しく読み込まれていない

**対処法**:
1. `chrome://extensions/` で拡張機能が有効になっているか確認
2. ページをリロード（F5）
3. コンソール（F12）でエラーを確認

### 「データ収集中」から進まない

**原因**: 価格データが取得できていない

**対処法**:
1. コンソールで以下を実行して価格が取得できているか確認:
   ```javascript
   // DOMから価格を手動取得
   const selectors = ['.rate', '[class*="rate"]', '[class*="price"]'];
   for (const sel of selectors) {
     const els = document.querySelectorAll(sel);
     console.log(sel, Array.from(els).map(el => el.textContent));
   }
   ```

2. チャートが表示されている通貨ペアを確認
3. ページを再読み込み

### シグナルが「NEUTRAL」ばかり

**原因**: 価格変動が少ない、または市場が動いていない

**対処法**:
- より変動の大きい通貨ペア（BTC/USD等）を選択
- 市場が開いている時間帯に使用
- 最低50ポイント（50秒）のデータ収集を待つ

## ⚙️ 設定のカスタマイズ

### データ収集数を変更

[trend-analyzer-engine.js](trend-analyzer-engine.js:6) を編集:

```javascript
// デフォルト: 500ポイント
analyzer = new SimpleTrendAnalyzer({ maxSize: 500 });

// より長期分析: 1000ポイント
analyzer = new SimpleTrendAnalyzer({ maxSize: 1000 });

// 短期分析: 200ポイント
analyzer = new SimpleTrendAnalyzer({ maxSize: 200 });
```

### 価格取得間隔を変更

[theoption-analyzer.js](theoption-analyzer.js:268) を編集:

```javascript
// デフォルト: 1秒ごと
}, 1000);

// より頻繁: 500ミリ秒ごと
}, 500);

// ゆっくり: 2秒ごと
}, 2000);
```

### パネル位置を変更

[theoption-analyzer.js](theoption-analyzer.js:105) のCSSを編集:

```css
#theoption-analyzer-panel {
  position: fixed;
  top: 80px;     /* 上からの距離 */
  right: 20px;   /* 右からの距離 */
  /* 左に配置したい場合: left: 20px; */
}
```

## 📝 注意事項

### ⚠️ 重要
- **投資は自己責任**: このツールはあくまで分析の補助です
- **100%の的中率ではありません**: 市場は予測不可能な要素があります
- **デモ口座で練習**: 実際の資金を使う前に十分テストしてください

### 💡 推奨事項
- 信頼度60%以上のシグナルを参考にする
- 複数の時間足で確認する
- 重要な経済指標発表時は避ける
- 自分の分析と組み合わせて使う
