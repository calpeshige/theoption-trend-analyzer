**場所**: 時間枠情報エリア


- 行189-192: HTML要素追加

**メリット**:
- 次の更新タイミングが分かる
- 時間枠によって更新頻度が違うことを体感できる
- 「動いている感」が伝わる

## 🔧 技術的な実装詳細

### カウントダウン更新ロジック

```javascript
function updateCountdown(elapsed, interval) {
  const remaining = Math.max(0, Math.ceil(interval - elapsed));

  if (remaining === 0) {
    // 分析中
    countdownEl.textContent = '🔄 分析中...';
    countdownEl.style.color = '#4CAF50';
  } else if (remaining <= 3) {
    // 残り3秒以下
    countdownEl.textContent = `${remaining}秒`;
    countdownEl.style.color = '#FFA726';
  } else {
    // 通常
    countdownEl.textContent = `${remaining}秒`;
    countdownEl.style.color = '#e0e0e0';
  }
}
```

### 総合スコアバーの計算

```javascript
// -100～+100を0%～100%に変換
const scorePercent = ((multiDim.score + 100) / 200) * 100;

// 例: スコア +75 の場合
// (75 + 100) / 200 * 100 = 175 / 200 * 100 = 87.5%
```

### ADXの★評価ロジック

```javascript
let adxStars = '⭐';
if (adxValue > 60) adxStars = '⭐⭐⭐⭐⭐';
else if (adxValue > 40) adxStars = '⭐⭐⭐⭐';
else if (adxValue > 25) adxStars = '⭐⭐⭐';
else if (adxValue > 20) adxStars = '⭐⭐';
```

### ML精度ランク判定

```javascript
if (mlDataCount >= 3000) {
  rankStars = '⭐⭐⭐⭐';
  rankLabel = '最高精度';
  rankColor = '#FFD700';
} else if (mlDataCount >= 1000) {
  rankStars = '⭐⭐⭐';
  rankLabel = '高精度';
  rankColor = '#4CAF50';
} else if (mlDataCount >= 300) {
  rankStars = '⭐⭐';
  rankLabel = '中精度';
  rankColor = '#8BC34A';
} else if (mlDataCount >= 100) {
  rankStars = '⭐';
  rankLabel = '低精度';
  rankColor = '#FFA726';
}
```

### 次のランク計算

```javascript
let nextRankThreshold = 0;
let nextRankLabel = '';

if (mlDataCount < 300) {
  nextRankThreshold = 300;
  nextRankLabel = '⭐⭐ 中精度';
} else if (mlDataCount < 1000) {
  nextRankThreshold = 1000;
  nextRankLabel = '⭐⭐⭐ 高精度';
} else if (mlDataCount < 3000) {
  nextRankThreshold = 3000;
  nextRankLabel = '⭐⭐⭐⭐ 最高精度';
} else if (mlDataCount < 5000) {
  nextRankThreshold = 5000;
  nextRankLabel = '最大蓄積';
}

const remaining = nextRankThreshold - mlDataCount;
```
