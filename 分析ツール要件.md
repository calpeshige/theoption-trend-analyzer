# TheOption 分析ツール - 必要データ要件

## 📊 チャート形式の確認

TheOptionの実際のチャート:
- **エリアチャート** (青い塗りつぶし付きライン)
- ローソク足ではなく、**価格の推移ライン**
- HIGH/LOW 判定用のシンプルなバイナリーオプション向け表示

## 🎯 バイナリーオプション分析に必要な最小データ

### 1. **価格データ (Tick/時系列)**

これだけあれば分析可能:

```javascript
{
  time: 1729850400,      // Unix timestamp (秒)
  price: 111384,         // 現在価格
  bid: 111383,           // 買値 (オプション)
  ask: 111385            // 売値 (オプション)
}
```

**頻度**: 1秒ごと、またはTick更新ごと

### 2. **過去の価格履歴**

直近の価格データがあれば、以下の分析が可能:

```javascript
[
  { time: 1729850400, price: 111384 },
  { time: 1729850401, price: 111385 },
  { time: 1729850402, price: 111383 },
  // ... 過去N件
]
```

**必要な量**:
- **短期分析**: 直近100-500ポイント (1-5分)
- **中期分析**: 直近1000-3000ポイント (15分-1時間)
- **長期分析**: 直近5000-10000ポイント (数時間-1日)

## 🔍 実装可能なテクニカル分析

### ローソク足がなくても可能な分析

#### 1. **トレンド分析**
- **移動平均線 (MA)**: 5MA, 20MA, 50MA
- **EMA (指数移動平均)**: 短期・長期クロス
- **トレンドライン**: 価格の上昇・下降傾向

```javascript
// 直近N個の平均価格
function calculateMA(prices, period) {
  const sum = prices.slice(-period).reduce((a, b) => a + b, 0);
  return sum / period;
}

const ma5 = calculateMA(recentPrices, 5);
const ma20 = calculateMA(recentPrices, 20);

// ゴールデンクロス/デッドクロス判定
if (ma5 > ma20) {
  return "上昇トレンド (HIGH推奨)";
} else {
  return "下降トレンド (LOW推奨)";
}
```

#### 2. **モメンタム分析**
- **RSI (相対力指数)**: 買われすぎ・売られすぎ判定
- **MACD**: トレンド転換検出
- **ストキャスティクス**: 短期的な反転予測

```javascript
// RSI計算 (14期間)
function calculateRSI(prices, period = 14) {
  const changes = [];
  for (let i = 1; i < prices.length; i++) {
    changes.push(prices[i] - prices[i - 1]);
  }

  const gains = changes.map(c => c > 0 ? c : 0);
  const losses = changes.map(c => c < 0 ? -c : 0);

  const avgGain = gains.slice(-period).reduce((a, b) => a + b) / period;
  const avgLoss = losses.slice(-period).reduce((a, b) => a + b) / period;

  const rs = avgGain / avgLoss;
  const rsi = 100 - (100 / (1 + rs));

  return rsi;
}

const rsi = calculateRSI(recentPrices);

if (rsi > 70) return "買われすぎ → LOW推奨";
if (rsi < 30) return "売られすぎ → HIGH推奨";
```

#### 3. **ボラティリティ分析**
- **ボリンジャーバンド**: 価格変動幅
- **ATR (Average True Range)**: 値動きの大きさ
- **標準偏差**: 価格のばらつき

```javascript
// ボリンジャーバンド
function calculateBollingerBands(prices, period = 20, multiplier = 2) {
  const ma = calculateMA(prices, period);

  const squaredDiffs = prices.slice(-period).map(p => Math.pow(p - ma, 2));
  const variance = squaredDiffs.reduce((a, b) => a + b) / period;
  const stdDev = Math.sqrt(variance);

  return {
    upper: ma + (stdDev * multiplier),
    middle: ma,
    lower: ma - (stdDev * multiplier)
  };
}

const bb = calculateBollingerBands(recentPrices);
const currentPrice = recentPrices[recentPrices.length - 1];

if (currentPrice > bb.upper) return "バンド上限突破 → 反転の可能性 (LOW)";
if (currentPrice < bb.lower) return "バンド下限突破 → 反転の可能性 (HIGH)";
```

#### 4. **パターン認識**
- **ダブルトップ/ボトム**: 価格の天井・底
- **ヘッド&ショルダー**: 反転パターン
- **サポート/レジスタンスライン**: 価格の節目

```javascript
// 直近の高値・安値を検出
function findPeaksAndValleys(prices, threshold = 5) {
  const peaks = [];
  const valleys = [];

  for (let i = threshold; i < prices.length - threshold; i++) {
    const window = prices.slice(i - threshold, i + threshold + 1);
    const current = prices[i];

    if (current === Math.max(...window)) {
      peaks.push({ index: i, price: current });
    }
    if (current === Math.min(...window)) {
      valleys.push({ index: i, price: current });
    }
  }

  return { peaks, valleys };
}
```

### アプローチ1: シンプル版 (推奨)

```javascript
class SimpleTrendAnalyzer {
  constructor() {
    this.prices = [];  // 直近500ポイント保持
    this.maxSize = 500;
  }

  addPrice(price) {
    this.prices.push(price);
    if (this.prices.length > this.maxSize) {
      this.prices.shift();  // 古いデータを削除
    }
  }

  analyze() {
    if (this.prices.length < 50) {
      return { signal: 'WAIT', reason: 'データ不足' };
    }

    // MA分析
    const ma5 = this.calculateMA(5);
    const ma20 = this.calculateMA(20);

    // RSI分析
    const rsi = this.calculateRSI();

    // トレンド判定
    let signal = 'NEUTRAL';
    let confidence = 0;

    if (ma5 > ma20 && rsi < 70) {
      signal = 'HIGH';
      confidence = 70;
    } else if (ma5 < ma20 && rsi > 30) {
      signal = 'LOW';
      confidence = 70;
    }

    return {
      signal,
      confidence,
      indicators: { ma5, ma20, rsi },
      currentPrice: this.prices[this.prices.length - 1]
    };
  }

  calculateMA(period) {
    const sum = this.prices.slice(-period).reduce((a, b) => a + b, 0);
    return sum / period;
  }

  calculateRSI(period = 14) {
    // RSI計算ロジック
  }
}
```

### アプローチ2: 高度版 (TradingView風)

**必要なもの**: 価格データ + OHLC生成

価格Tickデータから疑似的なローソク足を生成:

```javascript
class CandleBuilder {
  constructor(timeframe = 60) {  // 60秒 = 1分足
    this.timeframe = timeframe;
    this.currentCandle = null;
    this.candles = [];
  }

  addTick(price, timestamp) {
    const candleTime = Math.floor(timestamp / this.timeframe) * this.timeframe;

    if (!this.currentCandle || this.currentCandle.time !== candleTime) {
      // 新しいローソク足を開始
      if (this.currentCandle) {
        this.candles.push(this.currentCandle);
      }

      this.currentCandle = {
        time: candleTime,
        open: price,
        high: price,
        low: price,
        close: price
      };
    } else {
      // 現在のローソク足を更新
      this.currentCandle.high = Math.max(this.currentCandle.high, price);
      this.currentCandle.low = Math.min(this.currentCandle.low, price);
      this.currentCandle.close = price;
    }
  }

  getCandles() {
    return this.candles;
  }
}
```

## 📊 データ取得の実装例

### Lightstreamer経由でTick価格を取得

```javascript
import { LightstreamerClient, Subscription } from 'lightstreamer-client-node';

const analyzer = new SimpleTrendAnalyzer();

// Lightstreamer接続
const client = new LightstreamerClient('https://push.theoption.com', 'MARKETSPULSE_REMOTE');
client.connect();

// BTC/USD (AssetId=1012) の価格を購読
const subscription = new Subscription(
  'MERGE',
  ['Asset_1012'],
  ['last', 'time']
);

subscription.addListener({
  onItemUpdate: (update) => {
    const price = parseFloat(update.getValue('last'));
    const time = parseInt(update.getValue('time'));

    // 分析エンジンに価格を追加
    analyzer.addPrice(price);

    // 分析実行
    const result = analyzer.analyze();
    console.log('シグナル:', result.signal);
    console.log('信頼度:', result.confidence);
    console.log('MA5:', result.indicators.ma5);
    console.log('MA20:', result.indicators.ma20);
    console.log('RSI:', result.indicators.rsi);
  }
});

client.subscribe(subscription);
```

## ✅ 結論: 必要な最小データ

バイナリーオプション分析ツールには、**ローソク足は不要**です。

### 絶対必要:
- ✅ **価格 (price/last)** - リアルタイム
- ✅ **時刻 (time/timestamp)** - 各価格ポイント

### あると便利:
- 📊 bid/ask - スプレッド確認
- 📊 volume - 出来高分析
- 📊 high/low - 当日レンジ

### 不要:
- ❌ OHLC (Open/High/Low/Close) - 自分で生成可能
- ❌ ローソク足 - ライン表示で十分

**実装優先度**:
1. Lightstreamer WebSocketで `last` 価格を取得
2. 配列に蓄積して移動平均・RSIを計算
3. HIGH/LOW判定ロジック実装
4. Chrome拡張機能のUIに表示

これで十分なトレンド分析が可能です!
