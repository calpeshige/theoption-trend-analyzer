| ADX値 | 強度ラベル | ★評価 |

| 10-20 | 弱い | ⭐ |

| 25-40 | 強い | ⭐⭐⭐ |
| 40-60 | 非常に強い | ⭐⭐⭐⭐ |

### 改善3: 統合判定システム

**4パターンの判定ロジック**:

#### パターン1: トレンド明確（推奨）
```
条件: ADX ≥ 25 かつ |score| ≥ 40
判定: 🟢 上昇トレンド明確 or 🔴 下降トレンド明確
推奨: エントリー推奨
```

#### パターン2: 方向不明瞭（注意）
```
条件: ADX ≥ 25 かつ |score| < 40
判定: ⚠️ 方向不明瞭
推奨: 慎重にエントリー
```

#### パターン3: 弱いトレンド（注意）
```
条件: ADX < 25 かつ |score| ≥ 40
判定: ⚠️ 上昇傾向だが弱い or 下降傾向だが弱い
推奨: 慎重にエントリー
```

#### パターン4: レンジ相場（様子見）
```
条件: ADX < 25 かつ |score| < 40
判定: ⚪ レンジ相場
推奨: 様子見推奨（トレンド不明確）
```

**実装コード**:
```javascript
if (adxValue >= 25 && Math.abs(multiDim.score) >= 40) {
  // パターン1: トレンド明確
  overallJudgment = `${trendDirection}トレンド明確`;
  judgmentIcon = trendDirection === '上昇' ? '🟢' : '🔴';
} else if (adxValue >= 25) {
  // パターン2: 方向不明瞭
  overallJudgment = '方向不明瞭';
  judgmentIcon = '⚠️';
} else if (Math.abs(multiDim.score) >= 40) {
  // パターン3: 弱いトレンド
  overallJudgment = `${trendDirection}傾向だが弱い`;
  judgmentIcon = '⚠️';
} else {
  // パターン4: レンジ相場
  overallJudgment = 'レンジ相場';
  judgmentIcon = '⚪';
}
```

**メリット**:
- 方向と強度の両方を考慮
- 矛盾が解消される
- エントリー判断が明確

### 改善5: タブ切り替え時の再分析を無効化

**Before（問題）**:
```
15秒タブ: HIGH 65%
 ↓ タブ切り替え
5分タブ: 見送り 45% ← 数値が変わる！
```

**After（修正後）**:
```
15秒タブ: HIGH 65%
 ↓ タブ切り替え
5分タブ: HIGH 65% ← 変わらない（次回更新まで待つ）
```

**実装変更**:

**修正前**:
```javascript
function switchTimeframe(newTimeframe) {
  currentTimeframe = newTimeframe;
  performAnalysis(currentPrice, { isTabSwitch: true }); // ← 即座に再分析
}
```

**修正後**:
```javascript
function switchTimeframe(newTimeframe) {
  currentTimeframe = newTimeframe;
  // タブ切り替え時は再分析しない（次回の定期更新まで待つ）
  console.log('次回の定期更新で分析を実行します');
}
```

**メリット**:
- タブ切り替えで数値が変わらない
- ユーザーの混乱を防ぐ
- 次回の定期更新（10-60秒後）で新しい時間枠用の分析が実行される

## 📚 用語説明

### トレンド方向
- 各テクニカル指標（MACD, Stochastic, ROC等）の総合的な方向性
- 上昇・中立・下降の3段階

### トレンド強度（ADX）
- トレンドの一貫性・明確さを示す指標
- 方向は含まない（上昇でも下降でも強度は同じ）
- 0-100の範囲（25以上で明確なトレンド）

### 総合判定
- トレンド方向とトレンド強度を統合した最終判定
- エントリーの推奨度を示す

### テクニカル分析
- リアルタイムのテクニカル指標による分析
- 30秒分のデータがあれば利用可能
- 即座に結果が出る

### AI予測（機械学習）
- 過去の取引結果パターンから学習
- 100件以上のデータが必要
- 精度は件数に応じて向上（最大5000件）
