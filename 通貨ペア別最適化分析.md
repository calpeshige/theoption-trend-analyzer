# 通貨ペア別の閾値最適化分析

## 実装された自動最適化の仕組み

現在の実装は**ATRベースの自動調整**により、理論上は**全通貨ペアで自動対応**します。

### ATR正規化の原理

```javascript
// 各通貨ペアのATRは価格レンジに応じて異なる
BTC/JPY (154円):   ATR = 0.0362%
EUR/USD (1.05ドル): ATR = 0.020%
GBP/JPY (190円):   ATR = 0.080%

// 閾値計算（300秒判定の例）
閾値 = ATR × 0.90

BTC/JPY: 0.0362% × 0.90 = 0.0326% ≈ 0.05円
EUR/USD: 0.020% × 0.90 = 0.018% ≈ 0.00019ドル
GBP/JPY: 0.080% × 0.90 = 0.072% ≈ 0.14円

→ 価格レンジが違っても「同じ感度」で動作
```

### 低ボラティリティ通貨（EUR/GBP）

```

15秒閾値 = 0.015% × 0.09 = 0.00135%
300秒閾値 = 0.015% × 0.90 = 0.0135%

→ 小さな動きにも反応（適切）
```

### 中ボラティリティ通貨（EUR/USD, USD/JPY）

```
→ バランスの取れた反応（適切）
```

```
→ 大きな動きだけ反応（適切）
```

## 潜在的な問題と対策

### ⚠️ 問題1: 極端に低ボラティリティな通貨

**症状**:
- ATR < 0.010%（極めて低い）
- 閾値が小さすぎてノイズを拾う

**対策**:
```javascript
// detailed-segment-analyzer.js の minThreshold が機能
15秒: minThreshold = 0.0032%
→ ATRが小さくても最低この閾値は保証される
```

**該当通貨**: EUR/GBP（深夜時間帯）

### ⚠️ 問題2: 極端に高ボラティリティな通貨

**症状**:
- ATR > 0.15%（極めて高い）
- 閾値が大きすぎて動きを検出できない

**対策**:
現在は上限設定なし（ATRに完全追従）
→ 高ボラでも適切に動作するはず

**該当通貨**: GBP/JPY（重要指標発表時）

### ⚠️ 問題3: 価格レンジが極端に異なる通貨

**症状**:
- NZD/USD（0.6ドル）とGBP/JPY（190円）を同時使用
- パーセント表示は同じでも絶対値が大きく異なる

**対策**:
✅ すでに実装済み（ATR%ベース）
→ 価格レンジに関係なく動作

## 通貨ペア別の調整が必要なケース

通常は**調整不要**ですが、以下の場合は個別調整を検討：

### ケース1: 特定通貨のFLAT表示が多すぎる

**症状**: GBP/JPYで全セグメントFLAT → データが保存されない

**原因**: 閾値が高すぎる（ATRが大きい）

**対策**: 通貨別の係数を追加
```javascript
const currencyAdjustment = {
  'GBP/JPY': 0.7,  // 閾値を30%下げる
  'GBP/USD': 0.7
};

threshold = atrBasedThreshold * (currencyAdjustment[assetName] || 1.0);
```

### ケース2: 特定通貨のFLAT表示が出ない

**症状**: EUR/GBPで全セグメントUP/DOWN → ノイズを拾いすぎ

**原因**: 閾値が低すぎる（ATRが小さい）

**対策**: 通貨別の係数を追加
```javascript
const currencyAdjustment = {
  'EUR/GBP': 1.5,  // 閾値を50%上げる
};
```

```javascript
// Chrome DevToolsで実行
chrome.storage.local.get(null, (data) => {
  Object.keys(data).forEach(key => {
    if (key.startsWith('theoption_ml_')) {
      const asset = key.replace('theoption_ml_', '');
      const items = data[key];

      if (items.length > 0) {
        const latest = items[items.length - 1];
        const atr = latest.atrPercent;

        console.log(`\n${asset}:`);
        console.log(`  ATR: ${(atr * 100).toFixed(4)}%`);
        console.log(`  15秒閾値: ${(atr * 0.09 * 100).toFixed(4)}%`);
        console.log(`  60秒閾値: ${(atr * 0.36 * 100).toFixed(4)}%`);
        console.log(`  300秒閾値: ${(atr * 0.90 * 100).toFixed(4)}%`);

        // セグメント分析の確認
        const seg60 = latest.priceSegments60s;
        if (seg60 && seg60.segments) {
          const pattern = seg60.segments.map(s => s.direction).join('-');
          console.log(`  60秒パターン: ${pattern}`);
        }
      }
    }
  });
```
