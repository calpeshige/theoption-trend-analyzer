## 🔧 技術詳細

### 複合インデックスの仕組み

#### 1. モメンタムインデックス
```javascript
_classifyMomentum(rsi, stochastic) {
  const rsiScore = normalizeIndicator(rsi, 0, 100);      // 0-1
  const stochScore = normalizeIndicator(stochastic, 0, 100);  // 0-1
  const combined = rsiScore * 0.6 + stochScore * 0.4;  // 加重平均

  // 5段階分類
  if (combined >= 0.75) return "強い買われすぎ";
  if (combined >= 0.6) return "買われすぎ";
  if (combined <= 0.25) return "強い売られすぎ";
  if (combined <= 0.4) return "売られすぎ";
  return "中立";
}
```

#### 2. トレンドインデックス
```javascript
_classifyTrend(macdStrength, rocValue, maCross) {
  const macdScore = normalizeTrendIndicator(macdStrength, -0.03, 0.03);  // -1〜1
  const rocScore = normalizeTrendIndicator(rocValue, -0.8, 0.8);         // -1〜1

  // MA Crossのスコア
  let maCrossScore = 0;
  if (maCross.crossover === 'GOLDEN') maCrossScore = 0.8;
  else if (maCross.crossover === 'DEAD') maCrossScore = -0.8;
  else if (maCross.trend === 'UP') maCrossScore = 0.3;
  else if (maCross.trend === 'DOWN') maCrossScore = -0.3;

  // 加重平均
  const combined = macdScore * 0.4 + rocScore * 0.3 + maCrossScore * 0.3;

  // 5段階分類
  if (combined >= 0.6) return "強い上昇";
  if (combined >= 0.2) return "上昇";
  if (combined <= -0.6) return "強い下降";
  if (combined <= -0.2) return "下降";
  return "横ばい";
}
```

#### 3. 強度インデックス

```javascript
_classifyStrength(adxValue) {
  if (adxValue >= 35) return "強い";
  if (adxValue <= 20) return "弱い";
  return "中";
}
```

### 積集合演算による絞り込み

```javascript
// 各インデックスから候補を取得
const candidateSets = [
  indexes.segment["UP-UP-UP-DOWN-DOWN-DOWN"],    // 例: 500件
  indexes.momentum["買われすぎ"],                   // 例: 800件
  indexes.trend["上昇"],                           // 例: 600件
  indexes.strength["強い"]                         // 例: 400件
];

// 積集合（AND演算）
let matching = intersection(candidateSets);  // 例: 80件

// 適応的フィルタリング
if (matching.length < 30) {
  // 条件緩和: セグメント + モメンタムだけで再検索
  matching = intersection([candidateSets[0], candidateSets[1]]);
}

if (matching.length > 200) {
  // 条件追加: 強度フィルタを追加
  matching = matching.filter(idx => strengthSet.has(idx));
}

// 詳細な類似度計算（絞り込んだ80件だけ）
const withSimilarity = matching.map(data => ({
  pattern: data,
```

## 🐛 トラブルシューティング

### エラー: 「AI学習システムが初期化されていません」
**原因**: ページ読み込みが完了していない
**対処**: ページをリロードして、数秒待ってから再度実行

### エラー: 「データが少なすぎます」
**原因**: データ件数が100件未満
**対処**: もう少しデータを収集してから最適化を実行

### エラー: 「IndexedPatternMatcher が読み込まれていません」
**原因**: スクリプトの読み込み順序が間違っている
**対処**: manifest.jsonを確認し、indexed-pattern-matcher.jsがmachine-learning-system.jsより前にあることを確認

### 最適化後も遅い
**原因1**: インデックスが構築されていない
**対処**: ステータスが「✅ 最適化完了」になっているか確認

**原因2**: データ件数が極端に多い（10,000件以上）
**対処**: 古いデータを削除するか、インデックスを再構築
