# 修正完了：超短期取引用の感度調整システム

## 📋 解決した問題

**問題**: テクニカル分析の信頼度が常に30%台で固定され、全く変動しない

**根本原因**: 超短期取引（15秒～5分）では価格変動が非常に小さく、標準的な指標閾値では全ての指標が「ほぼ0」の強度値を返していた

---

## ✅ 実装した解決策

### 時間枠別の感度調整システム

各時間枠に応じて指標の感度を自動調整する係数を導入：

```javascript
getScaleFactor(timeframeSeconds) {
  if (timeframeSeconds <= 15) return 100;    // 15秒: 100倍（超高感度）
  if (timeframeSeconds <= 30) return 50;     // 30秒: 50倍（高感度）
  if (timeframeSeconds <= 60) return 20;     // 60秒: 20倍（中感度）
  if (timeframeSeconds <= 180) return 5;     // 3分: 5倍（低感度）
  return 1;                                   // 5分: 通常感度
}
```

**動作**:
- 15秒取引 → 100倍の感度で微細な価格変動を検出
- 5分取引 → 通常の感度で大きな変動のみ検出
- 時間枠が短いほど感度を上げることで、小さな変動でも適切な信号を出せるようにする

## 🎯 システムの特徴

### 1. 自動適応

- 時間枠を変更するだけで、自動的に最適な感度に調整
- ユーザーは何も設定する必要なし

### 2. 時間枠整合性

- 各時間枠で適切なスケールの信号が出る
- 15秒で反応しすぎたり、5分で反応しなさすぎることがない

### 3. 拡張性

- 新しい指標を追加する際も、`scaleFactor`を渡すだけで対応可能
- 時間枠を追加する場合も`getScaleFactor()`を修正するだけ

### 4. 一貫性

- 全ての指標が同じスケールで調整される
- 指標間のバランスが保たれる
