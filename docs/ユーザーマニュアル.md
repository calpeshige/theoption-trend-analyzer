# Theoption Trading System ユーザーマニュアル

**バージョン: 5.1.2**

TheOptionのトレーディングプラットフォーム専用のChrome拡張機能です。高度なテクニカル分析とAI機械学習を組み合わせて、超短期取引（15秒〜5分）の予測支援を行います。

## インストール方法

### 手順

1. 配布されたZIPファイル（`Theoption_Trading_System_v5.1.2.zip`）を解凍します
2. Chromeブラウザで `chrome://extensions/` を開きます
3. 右上の「デベロッパーモード」をオンにします
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします
5. 解凍したフォルダを選択します
6. 拡張機能が追加されたら完了です

### 初回起動

1. TheOptionのトレーディング画面（https://jp.theoption.com/trading）を開きます
2. Chrome右上の拡張機能アイコンをクリック
3. サイドパネルが開き、分析が自動的に開始されます

## 基本的な使い方

### システムの仕組み

本システムは2つの分析エンジンを搭載しています：

1. **テクニカル分析**: RSI、MACD、ストキャスティクス等の指標を使用したリアルタイム分析
2. **AI予測**: 過去の価格パターンを学習し、類似パターンから将来の動きを予測

### 分析の流れ

1. TheOptionでチャートを表示すると、自動的に価格データを収集開始
2. テクニカル指標を計算し、相場状況を判定
3. AI予測システムが学習データを蓄積（最初は50件以上必要）
4. 十分なデータが集まると、シグナルを表示

- **15秒**: 最短の取引時間
- **30秒**: 短期取引

- **5分**: やや長めの取引

テクニカル指標に基づく分析結果を表示します。

| 表示 | 意味 |
|------|------|
| **HIGH** | 上昇シグナル（緑色） |
| **LOW** | 下降シグナル（赤色） |
| **見送り** | エントリー非推奨（黄色） |
| **データ収集中** | 分析中 |

#### AI予測カード（右）

機械学習による予測結果を表示します。

| 表示 | 意味 |
|------|------|
| **HIGH** | 上昇予測（60%以上）（緑色） |
| **LOW** | 下降予測（60%以上）（赤色） |
| **統合↑/↓** | 複数時間枠で一致した予測 |
| **好条件↑/↓** | 高勝率パターン検出 |
| **高ボラ↑/↓** | ボラティリティ適応シグナル |
| **上昇傾向/下降傾向** | 20pt以上の差がある傾向 |
| **見送り** | エントリー非推奨 |
| **学習中** | データ収集中（50件未満） |
| **データ不足** | 予測に必要なデータが不足 |

#### 星評価（★）の意味

シグナルの信頼度を5段階で表示します。

| 星の数 | 信頼度 |
|--------|--------|
| ★★★★★ | 90%以上 - 非常に高い |
| ★★★★☆ | 80-89% - 高い |
| ★★★☆☆ | 70-79% - 中程度 |
| ★★☆☆☆ | 65-69% - やや低い |
| ★☆☆☆☆ | 60-64% - 最低ライン |

- **トレンド強度ゲージ**: 現在のトレンドの強さを可視化
- **ボラティリティ**: 価格変動の激しさ（高/中/低）
- **グレード判定**: A〜Cで相場のトレード適性を評価

### AI予測詳細カード

AI予測の詳細情報を表示します。

- **上昇/下降確率バー**: 予測確率を視覚的に表示
- **層別化インサイト**: 相場コンテキストやボラティリティによる補正情報
- **基本情報**: マッチパターン数、類似度などの詳細

### AI学習状況カード

機械学習システムの状態を表示します。

- **収集データ**: 現在蓄積されているデータ件数（最大50,000件）
- **学習レベル**: AIの学習進捗（%表示）
- **プログレスバー**: 視覚的な学習進捗

### クイック設定

#### 類似度閾値

AI予測で使用するパターンマッチングの類似度の最低値を設定します。

| 設定値 | 説明 |
|--------|------|
| **50%** | 多くのパターンを使用（シグナル多め、精度やや低め） |
| **70%** | バランス型（推奨） |
| **80%** | 高類似度のみ使用（シグナル少なめ、精度高め） |
| **90%** | 非常に高類似度のみ（シグナル極少、精度最高） |

#### データ範囲

AI予測で参照する学習データの範囲を設定します。

| 設定値 | 説明 |
|--------|------|
| **500** | 直近500件のデータを使用 |
| **1K** | 直近1,000件のデータを使用 |
| **2K** | 直近2,000件のデータを使用 |
| **全期間** | 蓄積した全データを使用（推奨） |

## シグナルの見方

### シグナル発出の条件

#### AI予測シグナル

1. **HIGHシグナル**: 上昇確率が60%以上
2. **LOWシグナル**: 下降確率が60%以上
3. **強化シグナル**: 複数の条件を満たした高信頼度シグナル
   - 統合シグナル: 複数の時間枠で同じ方向を予測
   - 好条件シグナル: 過去の高勝率パターンと一致
   - 高ボラシグナル: ボラティリティが高く、明確な方向性あり
4. **傾向シグナル**: 上昇/下降の差が20pt以上

#### 見送り条件

以下の場合は「見送り」となります：

- 同値率（引き分け率）が30%を超える場合
- 上昇/下降の差が20pt未満の場合
- 学習データが50件未満の場合

### テクニカル分析シグナル

テクニカル指標（RSI、MACD、ストキャスティクス等）の総合判定により、HIGH/LOW/見送りを表示します。

## エントリー方法

### 推奨エントリー手順

1. **両方のシグナルを確認**
   - テクニカル分析とAI予測の両方が同じ方向（HIGH同士またはLOW同士）の場合が最も信頼性が高い

2. **星評価を確認**
   - ★3つ以上（70%以上）を推奨
   - ★5つ（90%以上）は特に信頼度が高い

3. **相場状況を確認**
   - グレードがAまたはBの時が適切
   - ボラティリティが「高」の場合は値動きが大きい

4. **エントリータイミング**
   - カウントダウンを見て、取引時間内にエントリー
   - シグナルが表示されている間にエントリー

### エントリー時の注意点

- **両シグナル一致が基本**: テクニカルとAIが同じ方向の時のみエントリー
- **見送りは従う**: 「見送り」の場合はエントリーを控える
- **データ不足時は待つ**: 「学習中」「データ不足」の場合は十分なデータが集まるまで待つ
- **無理なエントリーをしない**: 自信がない場合は見送る

### 資金管理

- 1回のエントリー金額は資金の2-5%程度を推奨
- 連敗時はエントリー金額を減らす
- 1日の損失上限を設定する

## よくある質問

### Q: シグナルが表示されません

**A:** 以下を確認してください：
1. TheOptionのトレーディング画面を開いていますか？
2. 接続インジケーターが緑色になっていますか？
3. AI学習状況で「学習中」と表示されている場合は、50件以上のデータが集まるまでお待ちください

### Q: 「データ不足」と表示されます

**A:** AI予測には最低50件のデータが必要です。TheOptionでチャートを表示し続けることでデータが自動的に蓄積されます。

### Q: 通貨ペアを変更したらデータはどうなりますか？

**A:** 各通貨ペアごとに学習データが個別に保存されます。別の通貨ペアに切り替えると、その通貨ペア用のデータが使用されます。

### Q: データは保存されますか？

**A:** はい、ブラウザのストレージに自動保存されます。ブラウザを閉じても、拡張機能を削除しない限りデータは保持されます。

### Q: 予測精度はどのくらいですか？

**A:** 学習データ量や相場状況により変動しますが、適切な条件（両シグナル一致、★3以上）でエントリーすることで精度を高めることができます。

### Q: 複数のPCで使えますか？

**A:** 拡張機能は複数のPCにインストール可能ですが、学習データは各PC個別に蓄積されます。データを移行したい場合は「JSONエクスポート」と「JSONインポート」をご利用ください。

### Q: 仮想通貨ペアと法定通貨ペアで動作は違いますか？

**A:** はい、システムは通貨ペアの種類を自動判別し、それぞれに最適化された設定を適用します：
- **仮想通貨ペア**（BTC/JPY, BTC/USD, ETH/JPY, ETH/USD）: ボラティリティが高いため、より厳格なシグナル閾値を使用
- **法定通貨ペア**（EUR/USD, GBP/USD等）: 標準的な閾値を使用
