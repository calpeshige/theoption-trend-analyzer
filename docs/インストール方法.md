# TheOption Trend Analyzer - インストール方法

## 🚧 現在の状態

**バージョン**: 1.0.0 (API調査版)
**ステータス**: 開発中 - API調査フェーズ

この拡張機能は現在、TheOptionのAPIエンドポイントを調査するための**調査モード**です。実際の分析機能はAPI仕様が特定された後に実装されます。

---

## インストール手順

### 1. Chrome拡張機能の開発者モードを有効化

1. Google Chromeを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. 右上の「**デベロッパーモード**」をONにする

### 2. 拡張機能を読み込む

1. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
2. フォルダを選択:
   ```
   /Users/shige/Documents/拡張機能/theoption_trend
   ```
3. 「選択」をクリック

### 3. インストール完了の確認

- 拡張機能リストに「**TheOption Trend Analyzer**」が表示される
- アイコンがツールバーに追加される

## 使用方法（調査モード）

### API調査の手順

#### Step 1: TheOptionにログイン
1. https://jp.theoption.com/trading にアクセス
2. アカウントにログイン
3. トレーディング画面を開く

#### Step 2: デベロッパーツールを開く
1. **F12** キーを押す（または右クリック > 検証）
2. **Console** タブを選択

#### Step 3: チャートを操作
1. 通貨ペアを選択（例: EUR/USD）
2. 時間足を変更（1分、5分、15分等）
3. 異なる通貨ペアに切り替える

#### Step 4: APIコールを記録
コンソールに以下のような情報が表示されます：

```
[API監視] fetch呼び出し検出
URL: https://platformapi.theoption.com/Client.svc/...
Method: POST
Headers: {...}
Body: {...}

[API監視] レスポンス受信
Status: 200
Data: {...}
```

### 記録すべき情報

以下の情報を `API調査メモ.md` にコピーしてください：

#### ✅ チェックリスト
- [ ] ローソク足データ取得API URL
- [ ] リクエストメソッド (GET/POST)
- [ ] 必要なヘッダー (Authorization, Cookie等)
- [ ] リクエストパラメータ
  - [ ] アセットID/通貨ペアコード
  - [ ] 開始時刻 (from)
  - [ ] 終了時刻 (to)
  - [ ] 時間足 (interval/detalization)
- [ ] レスポンスJSON構造
- [ ] アセット一覧取得API（あれば）

## トラブルシューティング

### コンソールに何も表示されない

**原因**: APIコールがまだ発生していない

**解決策**:
1. チャートが完全に読み込まれるまで待つ
2. 通貨ペアや時間足を変更してみる
3. ページをリロードする

### 「拡張機能のエラー」が表示される

**原因**: manifest.jsonの設定が正しくない

**解決策**:
1. Chrome拡張機能ページで「エラー」をクリックして詳細を確認
2. エラーメッセージをコピーして報告

### TheOptionのページで動作しない

**原因**: URLパターンが一致していない

**確認事項**:
- URLが `https://jp.theoption.com/trading` または `https://theoption.com/trading` で始まっているか
- ページが完全に読み込まれているか
