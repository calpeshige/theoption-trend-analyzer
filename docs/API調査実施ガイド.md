## 🚀 実施手順（5-10分）

### Step 1: Chrome拡張機能をインストール

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択:
   ```
   /Users/shige/Documents/拡張機能/theoption_trend/
   ```

### Step 2: TheOptionにログイン

1. https://jp.theoption.com/trading にアクセス
2. アカウントでログイン
3. トレーディング画面が表示されるのを確認

### Step 3: デベロッパーツールを開く

1. `F12` キーを押す（またはメニュー → その他のツール → デベロッパーツール）
2. **Console** タブを選択
3. 以下のメッセージが表示されることを確認:
   ```
   [TheOption Analyzer] 🚀 拡張機能を読み込みました (調査版 v1.0.0)
   [TheOption Analyzer] 📡 ネットワーク監視を開始します
   ```

### Step 4: チャート操作でAPIを記録

以下の操作を順番に実行してください:

1. **通貨ペアを選択**
   - EUR/USD、USD/JPY など任意の通貨ペアをクリック

2. **時間足を変更**
   - 1分足 → 5分足 → 15分足 と切り替え

3. **コンソールを確認**
   - 色付きログが表示される:
     - 🟢 緑: `[API監視] fetch呼び出し検出`
     - 🔵 青: `[API監視] レスポンス受信`
     - 🟠 橙: `[API監視] XHR呼び出し検出`
     - 🟣 紫: `[API監視] XHR レスポンス受信`

### Step 5: 情報を記録

## 💡 調査のコツ

### うまくAPIが記録されない場合

1. **ページをリロード**
   - F5でページをリロードして、拡張機能が確実に読み込まれるようにする

2. **Network タブも確認**
   - DevTools → Network タブ
   - XHR または Fetch フィルタをON
   - `platformapi.theoption.com` へのリクエストを探す

3. **拡張機能の状態確認**
   - `chrome://extensions/` で拡張機能が有効か確認
   - エラーがあれば「詳細」ボタンで確認

### 記録すべきポイント

- ✅ **完全なURL** - クエリパラメータも含めて
- ✅ **全ヘッダー** - AuthorizationやCookie等
- ✅ **リクエストボディ** - POSTの場合
- ✅ **レスポンス構造** - JSONの階層と各フィールド名
- ✅ **タイムスタンプ形式** - UNIX時間 or ISO8601
- ✅ **複数の通貨ペア** - assetIdの対応関係
