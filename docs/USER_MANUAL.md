## 概要

**TheOption Trend Analyzer** は、TheOptionのバイナリーオプション取引を支援するChrome拡張機能です。

### 主な機能
- ✅ **リアルタイム価格分析** - 1秒ごとに最新価格を取得
- 📊 **テクニカル分析** - MACD、ADX、Stochastic、ATRなど複数指標を統合
- 🤖 **AI予測（機械学習）** - 過去の類似パターンから予測
- ⏱️ **複数時間枠対応** - 15秒、30秒、60秒、180秒、300秒の判定時間
- 💾 **データ自動保存** - 通貨ペア別にデータを記録・復元
- 🎯 **エントリー支援** - HIGH/LOWの方向性と信頼度を表示

### システム要件
- Google Chrome（最新版推奨）
- TheOptionアカウント
- 安定したインターネット接続

---

## インストール方法

### 1. 拡張機能の読み込み

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `theoption_trend` フォルダを選択

### 2. インストール確認

拡張機能一覧に「TheOption Trend Analyzer」が表示されればOK

### 3. 初回起動

1. TheOptionのサイト（https://trade.theoption.jp/）を開く
2. ログインして取引画面を表示
3. 右下に分析パネルが表示される

## 機能説明

### 1. 判定時間の選択

**画面上部のタブで切り替え**
```
[15秒] [30秒] [60秒] [180秒] [300秒]
```

**各時間枠の特徴:**
| 判定時間 | 更新頻度 | データ必要量 | 特徴 |
|---------|---------|-------------|------|
| 15秒 | 15秒ごと | 120秒 | 短期トレンド、変動大 |
| 30秒 | 30秒ごと | 150秒 | バランス型 |
| 60秒 | 60秒ごと | 180秒 | 安定性高い、推奨 |
| 180秒 | 180秒ごと | 300秒 | 中期トレンド |
| 300秒 | 300秒ごと | 420秒 | 長期トレンド |

**💡 推奨**: 初心者は60秒から開始

### 2. テクニカル分析

**使用している指標:**
- **MACD** - トレンドの強さと方向性
- **ADX** - トレンドの強度
- **Stochastic** - 買われすぎ/売られすぎ
- **ATR** - ボラティリティ（変動幅）
- **ROC** - 価格変化率
- **移動平均線** - 短期・中期・長期のトレンド
など

**判定の見方:**
```
LOW 68% ⬇
信頼度: ★★★☆☆
```
- **方向性**: HIGH（上昇）/ LOW（下降）/ 見送り
- **信頼度**: パーセンテージ（70%以上推奨）
- **星**: ★1つ = 20%、★5つ = 100%

### 3. AI予測（機械学習）

**仕組み:**
```
現在の相場状況
  ↓
過去の類似パターンを検索（最大５万件）
  ↓
類似度50%以上のパターンを抽出（類似度は選択可能）
  ↓
結果を統計分析
  ↓
HIGH/LOWを予測
```

**表示内容:**
```
🤖 AI予測（機械学習）
  HIGH 65% ⬆
  信頼度: ★★★☆☆

  類似パターン: 245件
  予測精度: 65.3%

  【AI予測根拠】
  ✓ 現在と似た相場で上昇が多い
  ✓ 類似度が高いパターンを発見
  ✓ 十分なデータ量
```

**重要ポイント:**
- 類似パターン数が**100件以上**で信頼性UP
- 予測精度が**70%以上**を推奨
- データが蓄積されるほど精度向上

### 4. データ収集

このシステムは**2種類のデータ**を収集しています：

#### 📈 テクニカル分析用データ（価格履歴）
```
【保存場所】メモリ（一時的）
【データ量】最大600秒分の価格履歴
【更新頻度】1秒ごと
【リセット条件】通貨ペア変更時
```

**特徴:**
- リアルタイム分析用
- 短期的なトレンド判定
- 即座に分析開始可能

#### 🤖 AI予測用データ（機械学習データ）
```
【保存場所】LocalStorage/Chrome Storage（永続化）
【データ量】無制限（最大50000件）
【更新頻度】15秒ごと
【リセット条件】拡張機能の再インストール時のみ
```

**特徴:**
- 通貨ペア別に保存
- ブラウザを閉じても保持
- 通貨ペア変更しても**リセットされない**
- データ蓄積で精度向上

#### データ収集の流れ
```
取引画面を開く
  ↓
【2秒後】価格データ収集開始
  ↓
【120秒後】テクニカル分析開始
  ↓
【15秒ごと】AI予測用データ蓄積
  ↓
【100件到達】AI予測開始
  ↓
【10000件以上】精度が安定
```

## 設定項目

**場所**: パネル下部のスライダー

**設定範囲**: 50% - 90%


**変更方法**:
1. ％を選択
2. 自動的に予測が再計算される
3. 結果を確認してエントリー

---

### データ期間（Data Limit）

**場所**: 設定メニュー

**選択肢**:
- 全期間（推奨）
- 直近1000件
- 直近500件

**使い分け**:
```
全期間:
  - データ4000件以上
  - 高精度を求める場合

直近1000件:
  - データ5000件以上
  - 最近の相場を重視

直近500件:
  - 処理速度優先
  - テスト用
```

## データ管理

**表示項目の意味:**
- **設定**: 検索範囲（全期間/直近1000件/直近500件）
- **総データ数**: 現在蓄積されているMLデータの件数
- **使用率**: 検索範囲に対する使用割合

#### データ件数の目安
```
100件未満     → AI予測は「データ不足」
100-500件     → 予測開始（精度は低め）
500-1000件    → 予測精度が向上
5000件   　　　→ 安定した精度
10000件以上    → 高精度・推奨
```

### データ容量の管理

#### ストレージ使用量の目安

```
データ件数別の使用量:
  1,000件   → 約2-3MB
  3,000件   → 約6-9MB
  5,000件   → 約10-15MB
  10,000件  → 約20-30MB
```

**LocalStorageの制限:**
- 超過すると古いデータから自動削除される可能性あり

**推奨管理方法:**
```
【データが3000件以上の場合】
→ そのまま使用（最適）

【データが10000件以上の場合】
→ 問題なし（高精度を維持）

【ストレージエラーが出る場合】
→ データ期間を「直近1000件」に変更
→ または拡張機能を再インストール
```

|------|------|------|

| **エントリー** | 取引を開始すること | HIGHまたはLOWを選択 |
| **HIGH** | 価格が上昇すると予測 | 上矢印⬆で表示 |

#### 分析関連用語

| 用語 | 意味 | 備考 |
|------|------|------|
| **テクニカル分析** | 過去の価格データから統計的に分析する手法 | MACD、ADX、Stochastic等を使用 |
| **AI予測** | 機械学習で過去の類似パターンから予測 | 100件以上のデータが必要 |

| **信頼度** | 予測の確からしさ | パーセンテージと★で表示 |
| **類似パターン数** | 現在と似た過去のパターンの件数 | 多いほど信頼性向上 |
| **予測精度** | AI予測の的中率 | 70%以上が理想的 |

#### テクニカル指標

| 用語 | 正式名称 | 意味 |
|------|---------|------|
| **MACD** | Moving Average Convergence Divergence | トレンドの方向性と強さを示す指標 |
| **ADX** | Average Directional Index | トレンドの強度を示す指標 |
| **Stochastic** | ストキャスティクス | 買われすぎ/売られすぎを判定 |
| **ATR** | Average True Range | 価格の変動幅（ボラティリティ）を示す |
| **ROC** | Rate of Change | 価格変化率を示す指標 |
| **MA** | Moving Average | 移動平均線、トレンドの方向性を示す |

| 用語 | 意味 | 備考 |
