## はじめに

**Theoption Trading System** は、TheOptionプラットフォームでの超短期取引（15秒〜5分）をサポートするChrome拡張機能です。

### 主な機能

| 機能 | 説明 |
|------|------|
| **テクニカル分析** | 複数の指標を組み合わせたリアルタイム分析 |
| **AI予測** | 過去の価格パターンから学習した予測 |
| **傾向シグナル** | 明確なシグナルに満たない場合でも方向性を表示 |
| **アラート機能** | シグナル発生時に音声で通知 |
| **5つの時間枠** | 15秒、30秒、60秒、3分、5分に対応 |

### 対応環境

- **ブラウザ**: Google Chrome（最新版推奨）
- **対応サイト**: TheOptionトレーディング画面
  - https://jp.theoption.com/trading

## インストール方法

### ステップ1: ZIPファイルの解凍

ダウンロードした `Theoption_Trading_System_v5.0.0.zip` を解凍します。

### ステップ2: Chrome拡張機能の読み込み

1. Chromeで `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」** をONにする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. 解凍したフォルダを選択

### ステップ3: インストール確認

拡張機能一覧に「Theoption Trading System」が表示されればインストール完了です。

### ステップ4: 使用開始

1. TheOptionのトレーディング画面を開く
2. 拡張機能のアイコンをクリック
3. **サイドパネル**が右側に表示される

### シグナルカード

2つのシグナルカードが横並びで表示されます：

```
┌──────────────────┬──────────────────┐
│ テクニカル分析   │    AI予測        │
│      ●           │      ●           │
│    HIGH          │     LOW          │
│  ★★★☆☆         │  ★★★★☆         │
└──────────────────┴──────────────────┘
```

- **上部**: 分析タイプ名
- **中央**: シグナルアイコン（色で方向を表示）
- **シグナル名**: HIGH / LOW / 上昇傾向 / 下降傾向 / 見送り / 学習中
- **星表示**: 信頼度（★1〜★5）

---

### 星表示の意味

**星の数は信頼度を表します：**

| 星の数 | 表示 | 信頼度 | 説明 |
|--------|------|--------|------|
| ★☆☆☆☆ | 1つ | 60-64% | シグナル基準ギリギリ |
| ★★☆☆☆ | 2つ | 65-69% | やや信頼できる |
| ★★★☆☆ | 3つ | 70-79% | 信頼できる |
| ★★★★☆ | 4つ | 80-89% | かなり信頼できる |
| ★★★★★ | 5つ | 90%以上 | 非常に高い信頼度 |

**傾向シグナルの場合：**

| 星の数 | 条件 |
|--------|------|
| ★☆☆☆☆ | 上昇と下降の差が20pt以上 |
| ★★☆☆☆ | 上昇と下降の差が30pt以上 |

## シグナルの種類と意味

### テクニカル分析シグナル

複数のテクニカル指標を統合して判定します。

| シグナル | 色 | 意味 | エントリー推奨 |
|----------|-----|------|----------------|
| **HIGH** | 🟢緑 | 上昇シグナル | HIGHでエントリー |
| **LOW** | 🔴赤 | 下落シグナル | LOWでエントリー |
| **見送り** | ⚪グレー | 明確な方向性なし | エントリー控える |

---

### AI予測シグナル

過去の類似パターンから予測します。

| シグナル | 色 | 意味 | エントリー推奨 |
|----------|-----|------|----------------|
| **HIGH** | 🟢緑 | 上昇率60%以上 | HIGHでエントリー |
| **LOW** | 🔴赤 | 下降率60%以上 | LOWでエントリー |
| **上昇傾向** | 🟢薄緑 | 上昇優勢（60%未満、差20pt+） | 参考程度 |
| **下降傾向** | 🔴薄赤 | 下降優勢（60%未満、差20pt+） | 参考程度 |
| **見送り** | ⚪グレー | 予測困難 | エントリー控える |
| **学習中** | ⚪グレー | データ収集中 | 待機 |

---

### 傾向シグナルについて

「上昇傾向」「下降傾向」は、60%の明確なシグナル基準には達していないものの、**方向性が見られる場合**に表示されます。

**表示条件：**
- 同値率（引き分け率）が30%以下
- 上昇率と下降率の差が20pt以上
- ただし、どちらも60%未満

**使い方：**
- 単独ではエントリーの根拠として弱い
- テクニカル分析と組み合わせて判断材料に
- 相場の方向性を把握する参考情報

## エントリー判断のガイドライン

### 推奨エントリー条件

#### 🏆 パターン1: ダブルシグナル一致（最も推奨）

```
テクニカル分析: HIGH ★★★☆☆以上
AI予測:         HIGH ★★★☆☆以上
           ↓
   HIGHでエントリー
```

**両方のシグナルが同じ方向で、星が3つ以上**の場合は最も信頼できます。

---

#### ⭐ パターン2: テクニカル高信頼度

```
テクニカル分析: HIGH ★★★★☆以上（80%+）
AI予測:         見送り または 上昇傾向
           ↓
   HIGHでエントリー（注意して）
```

テクニカル分析の星が**4つ以上**あれば、AIが見送りでも検討可能です。

---

#### ⭐ パターン3: AI高信頼度

```
テクニカル分析: 見送り
AI予測:         HIGH ★★★★☆以上（80%+）
           ↓
   HIGHでエントリー（注意して）
```

AI予測の星が**4つ以上**あれば、テクニカルが見送りでも検討可能です。

---

### エントリーを見送るべき条件

以下の場合は**エントリーを見送る**ことを強く推奨します：

| 状況 | 理由 |
|------|------|
| 両方とも「見送り」 | 明確な方向性がない |
| シグナルが逆方向 | テクニカルHIGH、AIがLOWなど |
| 星が1つしかない | 信頼度が低い |
| 傾向シグナルのみ | 確信度が不十分 |
| 同値率が高い | 引き分けリスクが高い |

---

### エントリーチェックリスト

エントリー前に必ず確認してください：

```
□ 時間枠を選択した
□ テクニカル分析を確認した（★★★以上推奨）
□ AI予測を確認した（★★★以上推奨）
□ 両方の方向性が一致している
□ 経済指標発表の直前・直後ではない
□ 資金管理ルールを確認した
□ 感情的になっていない

【全てチェックOK → エントリー】
【1つでもNG → 見送り】
```

---

### 時間枠ごとの注意点

| 時間枠 | 特徴 | 推奨星数 | アドバイス |
|--------|------|----------|-----------|
| **15秒** | ノイズが多い | ★★★★以上 | 高確信度のみエントリー |
| **30秒** | やや変動が激しい | ★★★以上 | 慎重に判断 |
| **60秒** | バランスが良い | ★★★以上 | 最も使いやすい |
| **3分** | 比較的安定 | ★★☆以上 | トレンド方向重視 |
| **5分** | 安定している | ★★☆以上 | 大きなトレンドに乗る |

## 設定項目の説明

### クイック設定（画面下部）

#### 類似度閾値

AI予測で過去パターンを検索する際の**類似度の最低基準**です。

```
[50%] [70%] [80%] [90%]
       ↑
    デフォルト
```

| 設定値 | 特徴 | 用途 |
|--------|------|------|
| **50%** | 広く検索 | パターン数を増やしたい時 |
| **70%**（推奨） | バランス型 | 通常使用 |
| **80%** | 厳格に検索 | 高精度を求める時 |
| **90%** | 非常に厳格 | 完璧なパターンのみ |

**アドバイス：**
- データが少ない時 → 50%で広く検索
- データが十分ある時 → 70%〜80%で精度重視

---

#### データ範囲

AI予測に使用する過去データの範囲です。

```
[500] [1K] [2K] [全期間]
                  ↑
               デフォルト
```

| 設定値 | 検索範囲 | 特徴 |
|--------|----------|------|
| **500件** | 直近500件のみ | 最新傾向重視、高速 |
| **1K件** | 直近1,000件 | バランス型 |
| **2K件** | 直近2,000件 | やや広め |
| **全期間**（推奨） | 全データ | 最も精度が高い |

**アドバイス：**
- 通常は「全期間」がおすすめ
- 相場急変時は「500件」で最新傾向を重視

## トレード戦略

### 初心者向け戦略

#### 戦略1: ダブルシグナル一致戦略（最推奨）

```
【エントリー条件】
✓ 時間枠: 60秒
✓ テクニカル: ★★★以上 & HIGH or LOW
✓ AI予測: ★★★以上 & 同じ方向
✓ 両方の方向性が一致

【手順】
1. 60秒タブを選択
2. テクニカルとAIを確認
3. 両方★★★以上 & 同じ方向 → エントリー
4. それ以外 → 見送り
```

**期待勝率**: 約60-65%

---

#### 戦略2: 高確信度単独戦略

```
【エントリー条件】
✓ テクニカル または AI が ★★★★以上（80%+）
✓ もう一方は見送りでもOK
✓ ただし逆方向はNG

【手順】
1. ★★★★以上のシグナルを確認
2. もう一方が逆方向でないことを確認
3. 高確信度のシグナル方向にエントリー
```

**期待勝率**: 約60-65%

---

### 中級者向け戦略

#### 戦略3: 複数時間枠確認戦略

```
【エントリー条件】
✓ 60秒と3分（180秒）の両方を確認
✓ 両時間枠で同じ方向性
✓ 両方とも★★以上

【手順】
1. 60秒タブでシグナル確認 → HIGH ★★★
2. 3分タブでシグナル確認 → HIGH ★★
3. 両方HIGH → 60秒でHIGHエントリー
4. 方向不一致 → 見送り
```

**期待勝率**: 約65-70%

---

#### 戦略4: 傾向シグナル活用戦略

```
【エントリー条件】
✓ テクニカル: HIGH or LOW ★★★以上
✓ AI予測: 同じ方向の「傾向」シグナル
  （上昇傾向 または 下降傾向）

【手順】
1. テクニカルで明確なシグナル確認
2. AIで同じ方向の傾向を確認
3. 両方同じ方向 → テクニカルの方向でエントリー
```

**期待勝率**: 約60-65%

---

### 資金管理の鉄則

```
【絶対に守るルール】

ルール1: 1回のエントリー上限
  → 総資金の 1-5% まで
  → 例: 10万円なら1回1,000-5,000円

ルール2: 1日の損失上限
  → 総資金の 10% まで
  → 上限に達したら即終了

ルール3: 連続負けの上限
  → 3回連続で負けたら30分〜1時間休憩
  → 冷静さを取り戻す

ルール4: 勝ち逃げ
  → 目標利益に達したら終了
  → 欲張らない
```

## よくある質問

### Q1. シグナルがなかなか出ない

**A**: 以下を確認してください：

1. **データ件数を確認** - AI学習状況で100件以上あるか
2. **類似度閾値を確認** - 90%だと厳しすぎる場合あり、70%に下げてみる
3. **相場状況** - 横ばい相場では方向性が出にくい

---

### Q2. AI予測が「学習中」のまま

**A**: AI予測には**最低100件の学習データ**が必要です。

- 新しい通貨ペアでは最初はデータが少ない
- トレーディング画面を開いたままにしてデータを収集
- 約1〜2時間でAI予測が開始される

---

### Q3. アラート音が鳴らない

**A**: 以下を確認してください：

1. 設定画面でアラート音が**ON**になっているか
2. ブラウザの音声が**ミュート**になっていないか
3. PCの音量が**0**になっていないか
4. Chromeの**サイト別音声設定**を確認

---

### Q4. テクニカルとAIで逆のシグナルが出る

**A**: **エントリーを見送ることを推奨します。**

- 両方が一致する場面を待つ
- または星が多い方を参考に（自己責任で）
- 相場の方向性が不明確な状態

---

### Q5. 勝率を上げるコツは？

**A**: 以下のポイントを意識してください：

1. **ダブルシグナル一致を待つ** - テクニカルとAIが同じ方向
2. **高い星数を狙う** - 最低★★★以上を推奨
3. **60秒時間枠を使う** - 最もバランスが良い
4. **見送りを恐れない** - 不確実な場面ではエントリーしない
5. **データを蓄積する** - 長く使うほどAI精度が向上

---

### Q6. データはどこに保存される？

**A**: ブラウザの**ローカルストレージ**に保存されます。

- 他の人には見えません
- ブラウザを閉じても保持されます
- **拡張機能を削除すると消えます**
- 定期的なバックアップを強く推奨

---

### Q7. 通貨ペアを変えたらデータはどうなる？

**A**: 通貨ペアごとに**独立して保存**されます。

- USD/JPY と EUR/JPY は別々のデータ
- 通貨ペアを切り替えても各データは保持される
- 切り替え後は該当通貨ペアのデータで分析される

---

### Q8. 経済指標発表時はどうすべき？

**A**: **発表前後5-10分はエントリーを避ける**ことを推奨します。

- 急激な価格変動が起きやすい
- 予測精度が低下する
- スリッページが発生しやすい

## トラブルシューティング

### サイドパネルが表示されない

**対処法：**

1. TheOptionの**トレーディング画面**にいることを確認
2. 拡張機能のアイコンをクリックしてサイドパネルを開く
3. ページを**再読み込み**（F5またはCtrl+R）
4. 拡張機能が**有効**になっているか確認（chrome://extensions/）

---

### 「データ収集中」から進まない

**必要なデータ量：**
- テクニカル分析: 約2分（120秒）
- AI予測: 100件（約50-100分）

**対処法：**
- そのまま待機してください
- データは自動的に蓄積されます

---

### 動作が重い・カクつく

**対処法：**

1. 他のChromeタブを閉じる
2. 不要な拡張機能を無効化
3. データ範囲を「500件」や「1K件」に変更
4. Chromeを再起動

---

### データが消えた

**考えられる原因：**
- 拡張機能を削除した
- ブラウザのデータを削除した
- Chromeをリセットした

**予防策：**
- 定期的に**JSONエクスポート**でバックアップ
- バックアップファイルを安全な場所に保存
