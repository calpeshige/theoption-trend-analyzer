// 修正前
const dataToSave = this.intelligentSampling(this.trainingData, 50000);

**効果:**

### 🔴 問題B: 類似度計算の欠陥 (CRITICAL)

**症状:**
```
[ML] 🔬 テクニカル指標: 58.0/60点 | RSI=15.0 MACD=15.0 ROC=12.0 MA=8.0 Stoch=5.0 ADX=3.0
[ML] 🔬 価格パターン: 37.0/40点 | トレンド=15/15(NEUTRAL⇄NEUTRAL) 傾き=10/10(Δ0.000)
```
- **全てのパターンが同じスコア**
- NEUTRAL同士で満点または高得点を獲得
- 結果として95-98%の類似度が大量発生

**根本原因:**
1. **価格パターンのトレンド一致**: NEUTRAL⇄NEUTRAL で満点15点
2. **テクニカル指標のトレンド一致**: NEUTRAL⇄NEUTRAL で40%の部分点

#### 修正1: 価格パターンのトレンド一致スコアリング (machine-learning-system.js:612-630)

```javascript
// 修正前
if (currentPattern.trendDirection === pastPattern.trendDirection) {
  trendPoints = 15;
  score += 15;
} else if (currentPattern.trendDirection === 'NEUTRAL' || pastPattern.trendDirection === 'NEUTRAL') {
  trendPoints = 7;
  score += 7;  // どちらかがニュートラルなら部分点
}
```

```javascript
// 修正後
if (currentPattern.trendDirection === pastPattern.trendDirection) {
  // NEUTRAL同士の場合は大幅減点（ほとんど意味のない一致）
  if (currentPattern.trendDirection === 'NEUTRAL') {
    trendPoints = 3;  // NEUTRAL⇄NEUTRAL: わずか3点のみ（15点 → 3点）
    score += 3;
  } else {
    // UP⇄UP または DOWN⇄DOWN: 満点
    trendPoints = 15;
    score += 15;
  }
} else if (currentPattern.trendDirection === 'NEUTRAL' || pastPattern.trendDirection === 'NEUTRAL') {
  // どちらかがNEUTRALで、もう一方がUP/DOWNの場合は0点
  trendPoints = 0;
  score += 0;
}
```

**変更内容:**
- NEUTRAL⇄NEUTRAL: **15点 → 3点** (80%減点)
- NEUTRAL⇄UP/DOWN: **7点 → 0点**
- UP⇄UP / DOWN⇄DOWN: **15点のまま** (変更なし)

#### 修正2: テクニカル指標のトレンド一致スコアリング (machine-learning-system.js:807-819)

```javascript
// 修正前
if (current.trend === past.trend) {

```

```javascript
// 修正後
if (current.trend === past.trend) {
  // NEUTRAL同士の場合は大幅減点
  if (current.trend === 'NEUTRAL') {
    score += maxPoints * 0.05; // NEUTRAL⇄NEUTRAL: わずか5%のみ（40% → 5%）
  } else {
    // UP⇄UP または DOWN⇄DOWN: 満点
    score += maxPoints * 0.4;
  }
} else if (current.trend === 'NEUTRAL' || past.trend === 'NEUTRAL') {
  // どちらかがNEUTRALで、もう一方がUP/DOWNの場合は0点
  score += 0;
}
```

**変更内容:**
- NEUTRAL⇄NEUTRAL: **40% → 5%** (87.5%減点)
- NEUTRAL⇄UP/DOWN: **20% → 0%**
- UP⇄UP / DOWN⇄DOWN: **40%のまま** (変更なし)

**影響を受ける指標:**
- RSI (15点満点): NEUTRAL⇄NEUTRALの場合 6.0点 → 0.75点
- MACD (15点満点): NEUTRAL⇄NEUTRALの場合 6.0点 → 0.75点
- ROC (12点満点): NEUTRAL⇄NEUTRALの場合 4.8点 → 0.6点

- **既存データの削除は不要**: 次回保存時に自動的に3000件に削減されます
- **類似度が大幅に低下**: これは正常な動作です。NEUTRAL同士の過剰評価が修正されました

この場合は、データ収集を継続して明確なトレンドのあるパターンを蓄積してください。
