これが完了しないと、次のフェーズに進めません。

### 調査に必要な情報
以下を `API調査メモ.md` に記録：
- [ ] エンドポイントURL（完全なパス）
- [ ] リクエストメソッド (GET/POST)
- [ ] 認証ヘッダー
- [ ] パラメータ（アセットID, from, to, detalization）
- [ ] レスポンスJSON構造

### Bubingaからの教訓
- 時刻に5分のマージンを設定（PC時刻ずれ対策）
- XMLHttpRequestを使用（fetchのフック回避）
- 確率を正規化（合計100%）
- Windows互換性（color-scheme: dark）
