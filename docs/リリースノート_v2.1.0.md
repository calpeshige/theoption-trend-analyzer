## ✨ 新しいUI

### コンパクト表示（常時表示）

ヘッダー部分（右上）に小さなボタンとして配置:

```
┌─────────────────────────┐
│ 📊 ⬆ HIGH      75%     │  ← クリックで詳細表示
└─────────────────────────┘
```

**位置**: ヘッダー右上（残高表示の左側）

**表示内容**:
- 📊 アイコン
- シグナル（HIGH/LOW/WAIT）
- 信頼度（%）

**色分け**:
- HIGH: 緑色テキスト
- LOW: 赤色テキスト
- WAIT/NEUTRAL: 白色テキスト

### ドロップダウン詳細（クリック時）

ボタンをクリックすると詳細パネルが下に展開:

```
┌─────────────────────────────┐
│       ⬆ HIGH               │
│      信頼度: 75%           │
├─────────────────────────────┤
│ MA5: 111384 │ MA20: 111380│
│ RSI: 65.23  │ 価格: 111384 │
├─────────────────────────────┤
│ 強い上昇トレンド | MA5 > MA20│
│ RSI 65 (中立-強気), ...    │
├─────────────────────────────┤
│           125pt             │
└─────────────────────────────┘
```

**閉じ方**:
- ボタンを再度クリック
- パネル外をクリック

## 📊 使い方

### 基本操作

1. **TheOptionにアクセス**: https://jp.theoption.com/trading
2. **ヘッダー確認**: 右上に「📊」ボタンが表示される
3. **待機**: 約50秒でデータ収集完了
4. **シグナル確認**: ボタンにHIGH/LOWが表示される
5. **詳細表示**: ボタンをクリックで詳細パネル展開

### シグナルの見方

#### ヘッダーボタン
- **⬆ HIGH (緑)**: 上昇トレンド → HIGHエントリー推奨
- **⬇ LOW (赤)**: 下降トレンド → LOWエントリー推奨
- **データ収集中**: まだデータ不足
- **NEUTRAL**: トレンド不明瞭 → 様子見推奨

#### 詳細パネル
- **シグナルボックス**: 大きく表示（HIGH/LOW）
- **信頼度**: 40-90%（60%以上推奨）
- **テクニカル指標**: MA5, MA20, RSI, 現在価格
- **判定理由**: どの指標が影響しているか
- **データポイント数**: 蓄積されたデータ量
