# テクニカル分析の信頼度計算の説明

## 📊 現在の計算ロジック

### 信頼度（Confidence）の計算方法

**場所**: [multi-indicator-system.js:308-326](multi-indicator-system.js#L308-L326)

```javascript
// 1. 各指標のスコアを重み付けして合計
totalScore =
  macdResult.strength * 2 +      // 20%
  stochasticResult.strength * 1.5 + // 15%
  rocResult.strength * 2 +        // 20%
  sentimentResult.strength * 2;   // 20%

maxScore = 20 + 15 + 10 + 20 + 20 = 100

// 2. 正規化（-100 ~ +100）
normalizedScore = (totalScore / maxScore) * 100

// 3. 信頼度の計算
if (normalizedScore > 50 && trendConfidence > 7) {
  signal = 'STRONG_HIGH';
  confidence = Math.min(95, 70 + normalizedScore / 3);
  // 例: normalizedScore = 75 → confidence = 70 + 75/3 = 95%
}
else if (normalizedScore > 30) {
  signal = 'HIGH';
  confidence = Math.min(85, 60 + normalizedScore / 4);
  // 例: normalizedScore = 40 → confidence = 60 + 40/4 = 70%
}
else if (normalizedScore < -50 && trendConfidence > 7) {
  signal = 'STRONG_LOW';
  confidence = Math.min(95, 70 + Math.abs(normalizedScore) / 3);
}
else if (normalizedScore < -30) {
  signal = 'LOW';
  confidence = Math.min(85, 60 + Math.abs(normalizedScore) / 4);
}
else {
  // ★ここが30%台の原因 ★
  signal = 'NEUTRAL';
  confidence = 30 + Math.abs(normalizedScore) / 2;
  // 例: normalizedScore = 15 → confidence = 30 + 15/2 = 37.5%
  // 例: normalizedScore = 0  → confidence = 30 + 0 = 30%
}
```

---

## 🔍 30%台になる原因

### 原因: normalizedScore が -30 ~ +30 の範囲

**NEUTRAL判定の条件**:
```
normalizedScore が -30 ~ +30 の範囲
→ 明確なトレンドがない
→ 信頼度 = 30 + |normalizedScore| / 2
→ 最低30%、最高45%
```

### 具体例

**ケース1: normalizedScore = 0（完全中立）**
```
各指標がバラバラ:
  MACD: +5
  Stochastic: -3
  ROC: +2
  Sentiment: -4

→ normalizedScore = 0
→ confidence = 30 + 0 / 2 = 30%
```

**ケース2: normalizedScore = 20（弱い上昇傾向）**
```
各指標がやや上昇:
  MACD: +8
  Stochastic: +5
  ROC: +6
  Sentiment: +7

→ normalizedScore = 20
→ confidence = 30 + 20 / 2 = 40%
```

**ケース3: normalizedScore = 35（HIGH判定）**
```
各指標が上昇:
  MACD: +10
  Stochastic: +8
  ROC: +9
  Sentiment: +8

→ normalizedScore = 35
→ signal = 'HIGH'
→ confidence = 60 + 35 / 4 = 68.75% ≈ 69%
```

## ⚠️ 問題点

### 問題1: NEUTRAL時の信頼度が低すぎる

**現状**:
```
normalizedScore = 0 → confidence = 30%
normalizedScore = 20 → confidence = 40%
```

**問題**:
- 30-40%の信頼度では、実質的に「使えない」判定
- ユーザーには「分析が機能していない」と見える
- 実際にはNEUTRAL（様子見）という正しい判定だが、信頼度が低すぎて誤解を招く

### 問題2: NEUTRAL判定が多すぎる可能性

**バイナリーオプションの特性**:
- 超短期取引（15秒-5分）
- 価格変動が小さい
- 明確なトレンドが出にくい

**結果**:
- normalizedScore が -30 ~ +30 に収まりやすい
- 常にNEUTRAL判定 → 信頼度30-40%
- 「テクニカル分析が役に立たない」と感じる

---

## 💡 改善案

### 改善案1: NEUTRAL時の信頼度の底上げ

```javascript
// 修正前
else {
  signal = 'NEUTRAL';
  confidence = 30 + Math.abs(normalizedScore) / 2;  // 30-45%
}

// 修正後
else {
  signal = 'NEUTRAL';
  confidence = 50 + Math.abs(normalizedScore) / 2;  // 50-65%
}
```

**効果**:
- NEUTRAL時でも50%以上の信頼度
- 「様子見推奨」という判定自体は正しいことを示す

### 改善案2: 判定基準の緩和

```javascript
// 修正前
else if (normalizedScore > 30) {
  signal = 'HIGH';
  confidence = Math.min(85, 60 + normalizedScore / 4);
}

// 修正後
else if (normalizedScore > 20) {  // 30 → 20 に変更
  signal = 'HIGH';
  confidence = Math.min(85, 55 + normalizedScore / 3);  // 調整
}
```

**効果**:
- normalizedScore が 20-30 の範囲でもHIGH判定
- より早く HIGH/LOW のシグナルが出る

### 改善案3: ADX依存度の軽減

```javascript
// 修正前
if (normalizedScore > 50 && trendConfidence > 7) {
  signal = 'STRONG_HIGH';
}

// 修正後
if (normalizedScore > 50 && trendConfidence > 5) {  // 7 → 5
  signal = 'STRONG_HIGH';
}
// または ADX条件を完全削除
if (normalizedScore > 50) {
  signal = 'STRONG_HIGH';
}
```

**効果**:
- ADXが低くても STRONG シグナルが出やすくなる
- トレンドが弱い相場でも判定が出る

### 改善案4: 複合的アプローチ（推奨）

```javascript
// シグナル判定
let signal, confidence;

if (normalizedScore > 50) {
  signal = 'STRONG_HIGH';
  confidence = Math.min(95, 75 + normalizedScore / 4);  // 87-100%
}
else if (normalizedScore > 20) {  // 30 → 20
  signal = 'HIGH';
  confidence = Math.min(85, 60 + normalizedScore / 3);  // 66-85%
}
else if (normalizedScore < -50) {
  signal = 'STRONG_LOW';
  confidence = Math.min(95, 75 + Math.abs(normalizedScore) / 4);
}
else if (normalizedScore < -20) {  // -30 → -20
  signal = 'LOW';
  confidence = Math.min(85, 60 + Math.abs(normalizedScore) / 3);
}
else {
  // NEUTRAL: -20 ~ +20
  signal = 'NEUTRAL';
  confidence = 50 + Math.abs(normalizedScore);  // 50-70%
}

// ADXボーナス（追加）
if (trendConfidence > 10) {
  confidence = Math.min(95, confidence + 5);  // +5%
}

// ボラティリティボーナス（追加）
if (volatilityBonus > 5) {
  confidence = Math.min(95, confidence + 3);  // +3%
}
```

**効果**:
- NEUTRAL範囲を -20 ~ +20 に狭める
- NEUTRAL時の信頼度を 50-70% に引き上げ
- ADX/ボラティリティをボーナスとして加算
- より実用的な信頼度に
