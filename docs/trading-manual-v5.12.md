## 1. はじめに

Theoption Trading Systemは、TheOptionのトレーディング画面と連動し、テクニカル分析とAI（機械学習）予測を組み合わせた取引支援ツールです。

**対応する判定時間：** 15秒・30秒・60秒・3分・5分

本システムは、サイドパネル上にリアルタイムでシグナルを表示し、エントリータイミングの判断をサポートします。

> **ご注意：** 本システムのシグナルは取引の参考情報です。投資判断はご自身の責任で行ってください。

## 2. インストールと起動

### インストール

1. Chrome ブラウザで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. 本システムのフォルダを選択

### 起動方法

1. TheOptionのトレーディング画面（`jp.theoption.com/trading`）を開く
2. ブラウザ右上の拡張機能アイコンをクリック
3. サイドパネルが自動的に開き、分析が開始されます

### 待機画面

TheOptionのトレーディング画面以外を表示している場合、サイドパネルには「待機中」画面が表示されます。トレーディング画面に移動すると自動的に分析が開始されます。

## 3. 画面構成

サイドパネルは上から順に以下の構成です。

### トップバー

| 要素 | 説明 |
|------|------|
| **接続ステータス** | 緑のドット＝接続中、テキストで状態を表示 |
| **データ件数** | 現在のAI学習データの総件数 |
| **通貨ペア名** | 現在選択中の通貨ペア（自動検出） |
| **カウントダウン** | 次の分析までの円形プログレスリング |
| **ダウンロードボタン** | データのダウンロード・バックアップ画面を開く |
| **設定ボタン（⚙）** | 各種設定画面を開く |

### 判定時間セレクター

5つのボタンから判定時間を選択します。

- **15秒** / **30秒** / **60秒** / **3分** / **5分**

選択中のボタンが強調表示されます。TheOptionの取引画面で選択している判定時間に合わせてください。

### シグナルカード

2つのカードが並んで表示されます。

- **テクニカル分析** — 複数の指標を統合した総合判断
- **AI予測** — 機械学習による予測結果

### 急変検出アラート

急激な価格変動を検出した場合、シグナルカードの下に黄色い「⚠ 急変検出」アラートが表示されます。急変時はシグナルの信頼性が低下するため、エントリーを控えることを推奨します。

### 20インジケータ多数決パネル（多数決モード時のみ）

多数決モードで使用される20個のインジケータの詳細を確認できます。ヘッダーをクリックすると展開/折りたたみが切り替わります。

### マーケット概況カード

相場の勢い・価格位置・ボラティリティ・予測文を表示します。

### AI予測詳細カード

AI予測の確率や層別化分析の詳細を表示します。

### AI学習状況カード

AIの学習データ量や学習レベル、データ管理機能を提供します。

### クイック設定

分析モード・類似度閾値・データ範囲をワンタッチで変更できます。

## 4. シグナルの見方

### シグナルの種類

| シグナル | 表示色 | 意味 |
|----------|--------|------|
| **HIGH** | 緑 | 上昇予測 → HIGHエントリー推奨 |
| **LOW** | 赤 | 下降予測 → LOWエントリー推奨 |
| **見送り** | グレー | 方向性が不明確 → エントリー非推奨 |
| **データ収集中** | グレー | 分析に必要なデータを収集中（カウントダウン表示） |

### 信頼度の表示

各シグナルカードの下部に信頼度が表示されます。信頼度が高いほど、シグナルの精度が期待できます。

### カウントダウン表示

分析開始前は「あと○分○秒」というカウントダウンが表示されます。この時間が経過すると、最初のシグナルが出力されます。

## 5. シグナルモード（標準 / 多数決）

本システムには2つのシグナルモードがあります。設定パネルから切り替え可能です。

### 多数決モード（デフォルト）

- テクニカル分析カードのヘッダーに **「多数決」** バッジ（青）が表示
- 20種類のインジケータで多数決を取り、13個以上が一致した方向をシグナルとして出力
- より多くのデータを必要とするため、分析開始までの待ち時間が長い
- 「20インジケータ多数決」パネルが表示され、各インジケータの判定を確認可能

**分析開始までの目安時間（多数決モード）：**

| 判定時間 | 待ち時間の目安 |
|----------|---------------|
| 15秒 | 約7分30秒 |
| 30秒 | 約15分 |
| 60秒 | 約30分 |
| 3分 | 約45分 |
| 5分 | 約75分 |

### 標準モード

- テクニカル分析カードのヘッダーに **「標準」** バッジ（オレンジ）が表示
- 複数のテクニカル指標を統合した総合分析方式
- 必要データ量が少なく、分析開始が早い
- 「20インジケータ多数決」パネルは非表示

**分析開始までの目安時間（標準モード）：**

| 判定時間 | 待ち時間の目安 |
|----------|---------------|
| 15秒 | 約2分 |
| 30秒 | 約5分 |
| 60秒 | 約8分 |
| 3分 | 約9分 |
| 5分 | 約10分 |

### どちらのモードを使うべきか？

| 状況 | 推奨モード |
|------|-----------|
| すぐにシグナルが欲しい | 標準モード |
| じっくり待てる・精度重視 | 多数決モード |

### モードの切り替え方

1. 右上の ⚙（設定）ボタンをタップ
2. 「シグナルモード」のドロップダウンから選択
3. 設定は即座に反映されます

> **注意：** モードを切り替えると、データ収集のカウントダウンがリセットされる場合があります。

## 6. マーケット概況

マーケット概況カードでは、現在の相場状況を一目で把握できます。カードヘッダーをタップすると展開/折りたたみが切り替わります。

### モメンタム（勢い）

相場の勢いの方向と強さを表示します。

| 表示 | 意味 |
|------|------|
| **↑ 上昇** | 上昇の勢いがある |
| **↓ 下降** | 下降の勢いがある |
| **➡ 横ばい** | 明確な方向性がない |

モメンタムバーで勢いの強さを視覚的に確認できます。

### ボラティリティバッジ

カードヘッダーに相場の変動幅を示すバッジが表示されます。

| バッジ | 意味 |
|--------|------|
| **高ボラ** | 値動きが大きい（チャンスが多いが注意も必要） |
| **低ボラ** | 値動きが小さい（シグナルが出にくい） |
| 表示なし | 通常の変動幅 |

### 価格位置

現在の価格が直近のレンジの中でどの位置にあるかを表示します。

### 予測文

現在の相場状況を総合した予測テキストが表示されます。

## 7. AI予測詳細

AI予測詳細カードでは、機械学習による予測の詳細情報を確認できます。

### 確率バー

上昇確率と下降確率が横棒グラフで表示されます。

- **緑（上昇）** — 上昇する確率
- **赤（下降）** — 下降する確率

確率の差が大きいほど、予測の方向性が明確です。

### 時間帯バッジ

AI予測が「時間帯別」モードで動作している場合、バッジが表示されます（クイック設定の「分析モード」と連動）。

### 層別化分析

AIは過去のデータを類似パターンで層別化（グループ分け）し、精度を高めています。層別化の結果がカード内に表示される場合があります。

### 基本情報

AIの予測に使われた学習データ件数やパターン一致度などの基本情報が表示されます。

## 8. AI学習状況とデータ管理

### AI学習状況カード

AI（機械学習）の学習進捗を確認できます。

| 項目 | 説明 |
|------|------|
| **収集データ** | 現在蓄積されているデータ件数（上限25,000件/通貨ペア） |
| **学習レベル** | AIの予測精度の目安（%表示） |
| **プログレスバー** | データ量の視覚的な進捗表示 |

### データ鮮度

データの新しさを星（★）で5段階評価します。古いデータが多いと精度が低下する場合があります。

### データ整理

上限を超えた古いデータを自動的に整理する機能です。「データ整理」ボタンが表示された場合、タップして古いデータを削除できます。整理中は画面がブロックされ、完了まで操作できません。

### 通貨ペア別データ状況

「📊 通貨ペア別データ状況」ボタンをタップすると、通貨ペアごとのデータ蓄積状況を一覧で確認できます。

各通貨ペアをタップして展開すると、時間帯別のデータ件数が表示されます。不要な時間帯のデータを選択して削除することも可能です。

## 9. クイック設定

メイン画面の下部に、よく使う設定を素早く変更できるクイック設定があります。

### 分析モード

| モード | 説明 |
|--------|------|
| **全体** | すべての時間帯のデータを使ってAI予測 |
| **時間帯別** | 現在と同じ時間帯のデータのみを使ってAI予測（精度向上が期待できる） |

**使い分け：**
- データが十分にある場合 → 「時間帯別」で精度向上を狙う
- データが少ない場合 → 「全体」で予測を安定させる

### 類似度閾値

AI予測時に、過去の類似パターンを探す際の一致度の基準です。

| 閾値 | 特徴 |
|------|------|
| **50%** | 広く一致を取る（データ多、精度やや低） |
| **70%**（推奨） | バランス型 |
| **80%** | やや厳密（精度高、一致数減少） |
| **90%** | 非常に厳密（精度最高、一致数がかなり少ない） |

データが十分にある場合は80%以上、少ない場合は50〜70%を推奨します。

### データ範囲

AI予測に使用する学習データの範囲を指定します。

| 範囲 | 説明 |
|------|------|
| **500** | 直近500件のみ使用 |
| **1K** | 直近1,000件のみ使用 |
| **2K** | 直近2,000件のみ使用 |
| **全期間**（推奨） | すべてのデータを使用 |

データが古くて精度が低いと感じる場合は、範囲を絞ると改善する場合があります。

画面全体の文字の大きさを「小」「中」「大」から選択します。

| 設定値 | 説明 |
|--------|------|
| **OFF** | フィルタなし（すべてのシグナルを表示） |
| **弱** | 明らかにトレンド終息時のみ抑制 |
| **中**（推奨） | バランスの良い抑制 |
| **強** | 積極的に抑制（シグナル数が減る） |

「見送り」シグナルが多すぎると感じたら「弱」に、騙しシグナルが多いと感じたら「強」にしてください。

## 11. エントリーガイド

### 基本的なエントリーフロー

```
① 判定時間を選択（TheOptionの画面と合わせる）
② カウントダウンが完了するまで待つ
③ テクニカル分析とAI予測のシグナルを確認
④ 両方が一致していればエントリー
```

### エントリー推奨条件

以下のすべてを満たす場合、最も精度の高いエントリーが期待できます。

1. **テクニカル分析とAI予測が同じ方向**（HIGH同士 or LOW同士）
2. **信頼度が高い**
3. **急変検出アラートが出ていない**
4. **モメンタムが明確な方向を示している**

### エントリーすべきでない状況

| 状況 | 理由 |
|------|------|
| シグナルが「見送り」 | 方向性が不明確 |
| テクニカルとAIが逆方向 | 分析が矛盾している |
| 急変検出アラートが表示中 | 相場が不安定 |
| データ収集中（カウントダウン表示中） | 分析が未完了 |

### 時間帯による推奨

#### 取引に適した時間帯

| 時間帯 | 市場 | 特徴 |
|--------|------|------|
| 9:00〜11:00 | 東京市場 | 比較的安定した値動き |
| 16:00〜18:00 | ロンドン市場開始 | 活発な値動き |
| 22:00〜24:00 | NY市場 | 大きな値動き |

#### 避けるべきタイミング

- 重要な経済指標の発表前後30分
- 市場の切り替わり時間帯
- 深夜〜早朝（値動きが少なく予測が不安定）

### 取引のDO / DON'T

**DO（推奨）**
- シグナルが一致したときのみエントリー
- カウントダウン完了後にシグナルを確認
- マーケット概況カードで相場の勢いを確認
- 連続で負けたら一旦休む
- データが十分に蓄積されてからAI予測を参考にする

**DON'T（非推奨）**
- シグナルを無視した感覚トレード
- 急変検出アラート中のエントリー
- データ収集中のエントリー
- 連続負け後の感情的なエントリー
- テクニカルとAIが矛盾しているときのエントリー

## 13. 時間帯別データ管理

特定の時間帯のデータを選択して削除する機能です。

### アクセス方法

1. AI学習状況カードの「📊 通貨ペア別データ状況」をタップ
2. 通貨ペアの一覧が表示される
3. 各通貨ペアをタップして展開
4. 時間帯別（0〜23時）のデータ件数が表示される

### データの削除

1. 削除したい時間帯のチェックボックスを選択
2. 「削除」ボタンをタップ
3. 確認ダイアログで「削除する」を選択

> **注意：** 削除したデータは元に戻せません。削除前にJSONエクスポートでバックアップを取ることを推奨します。

### 活用方法

- 取引しない時間帯のデータを削除することで、AI予測の精度向上が期待できます
- 例：深夜帯（1:00〜6:00）に取引しない場合、その時間帯のデータを削除

## 14. よくある質問

### Q: シグナルが出るまでにどのくらいかかりますか？

A: シグナルモードと判定時間により異なります。「あと○分○秒」のカウントダウンが完了すると最初のシグナルが表示されます。すぐにシグナルが欲しい場合は「標準モード」をお試しください。

### Q: 多数決モードと標準モード、どちらが精度が高いですか？

A: 相場状況により異なります。多数決モードは多くの指標で合意を取るため安定性が高い傾向です。標準モードは複合的な分析を行い、起動が早い利点があります。まずは多数決モードで使用し、待ち時間が気になる場合に標準モードをお試しください。

### Q: テクニカルとAI、どちらを優先すべきですか？

A: 両方が一致したときのみエントリーするのが最も安全です。片方だけで判断するのは推奨しません。

### Q: AI予測が「準備中」のまま表示されません

A: AI予測には一定量の学習データが必要です。初回利用時はデータが蓄積されるまで待つ必要があります。AI学習状況カードでデータの蓄積具合を確認してください。

### Q: 急変検出アラートが表示されたらどうすればいいですか？

A: 急激な価格変動が検出された状態です。シグナルの信頼性が低下しているため、アラートが消えるまでエントリーを控えることを推奨します。

### Q: データの上限に達した場合はどうなりますか？

A: 各通貨ペアにつき最大25,000件のデータが保存されます。上限に近づくと「データ整理」ボタンが表示されます。タップすると古いデータが自動的に削除され、新しいデータのための容量が確保されます。

### Q: 20インジケータ多数決パネルが表示されません

A: シグナルモードが「標準モード」になっている可能性があります。「多数決モード」に切り替えると表示されます。

### Q: シグナルが頻繁に変わるのですが？

A: 相場が不安定な状態です。シグナルが安定するまで待つか、モメンタムフィルタの強度を上げることをお勧めします。

### Q: ページをリロードするとカウントダウンが最初からになりますか？

A: はい、ページをリロードするとデータ収集が最初からやり直しになります。分析中はリロードを避けてください。

### Q: 別の通貨ペアに切り替えるとどうなりますか？

A: 通貨ペアを切り替えると、その通貨ペアのデータ収集が新たに開始されます。カウントダウンが再表示されます。
