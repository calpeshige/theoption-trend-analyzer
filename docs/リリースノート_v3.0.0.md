# TheOption Trend Analyzer v3.0.0 リリースノート

## 🎉 メジャーアップデート: 安定版マルチタイムフレーム分析

### 📍 変更内容

**v2.1.0 → v3.0.0**: **シグナル安定性の大幅改善** - 1秒ごとに変わる不安定なシグナルから、マルチタイムフレーム分析と3回連続確認システムによる安定したシグナルへ

---

## ✨ 主要な新機能

### 1. マルチタイムフレーム分析

リアルタイムの価格データから複数の時間足を生成し、総合的に分析:

- **1分足**: 短期トレンドの把握
- **5分足**: 中期トレンドの確認
- **15分足**: 長期トレンドの方向性

```
┌─────────────────────────────┐
│  1分: ⬆ HIGH               │
│  5分: ⬆ HIGH               │
│ 15分: ⬆ HIGH               │
└─────────────────────────────┘
```

**メリット**:
- 複数の時間軸から相場を俯瞰
- より信頼性の高いシグナル
- 短期のノイズに惑わされない

### 2. 3回連続確認システム

シグナルが3回連続で同じ方向を示した時のみ確定:

```
分析1回目: HIGH (信頼度70%) → 保留
分析2回目: HIGH (信頼度75%) → 保留
分析3回目: HIGH (信頼度72%) → ✅ 確定 (平均信頼度72%)
```

**効果**:
- シグナルのころころ変わる問題を解決
- 信頼度の高いエントリーポイントのみ表示
- 誤シグナルの大幅削減

### 3. トレンド継続時間の評価

トレンドがどれだけ持続しているかを評価し、信頼度に反映:

- **2分以上継続**: +5% 信頼度ボーナス
- **5分以上継続**: +10% 信頼度ボーナス
- **10分以上継続**: +15% 信頼度ボーナス

```
トレンド: 8分 → 強いトレンド確認 → 信頼度+10%
```

### 4. ボラティリティフィルター

価格変動が小さいレンジ相場ではシグナルを出さない:

- 最低0.1%の価格変動が必要
- レンジ相場での誤シグナルを防止

### 5. 強化された判定基準

より厳格なスコアリングシステム:

**旧版 (v2.x)**:
- スコア ≥2: HIGH
- スコア ≤-2: LOW

**新版 (v3.0)**:
- スコア ≥7: 強いHIGH (信頼度80-90%)
- スコア ≥5: HIGH (信頼度70-80%)
- スコア ≤-7: 強いLOW (信頼度80-90%)
- スコア ≤-5: LOW (信頼度70-80%)

## 🏗️ 技術的な実装

### 新しいコンポーネント

#### 1. CandleBuilder
リアルタイムの価格ティックからローソク足を生成:

```javascript
class CandleBuilder {
  constructor(timeframe = 60) // 60秒 = 1分足
  addTick(price, timestamp)   // 価格データ追加
  getCompletedCandles()       // 完成したローソク足取得
}
```

**特徴**:
- 1分、5分、15分の3つの時間足を同時生成
- OHLC (始値・高値・安値・終値) データ
- ページ読み込み時から蓄積開始

#### 2. SignalStabilizer
シグナルの3回連続確認:

```javascript
class SignalStabilizer {
  constructor(confirmCount = 3)
  addSignal(signal, confidence)

  // 戻り値:
  // { confirmed: true, signal: 'HIGH', confidence: 75 }
  // { confirmed: false, pending: 2, required: 3 }
}
```

**動作**:
- 直近3回の分析結果を保持
- すべて同じシグナルの場合のみ確定
- 信頼度は3回の平均値

#### 3. TrendContinuity
トレンドの持続時間を追跡:

```javascript
class TrendContinuity {
  updateTrend(signal, timestamp)
  getDuration() // 分単位で返す
  getConfidenceBonus() // 継続時間に応じたボーナス
}
```

#### 4. CandleAnalyzer
ローソク足データからテクニカル分析:

```javascript
class CandleAnalyzer {
  analyze(candles)

  // 計算指標:
  // - MA5, MA20, MA50
  // - RSI
  // - ボリンジャーバンド
  // - MACD
}
```

**改善点**:
- より厳格なスコアリング (±5以上)
- ボラティリティチェック追加
- 複数指標の総合判定

#### 5. MultiTimeframeAnalyzer
すべてのコンポーネントを統合:

```javascript
class MultiTimeframeAnalyzer {
  addTick(price, timestamp)
  analyze()

  // 戻り値:
  {
    signal: 'HIGH',
    confidence: 75,
    confirmed: true,
    status: '3回連続確認済み',
    timeframes: { '1m': 'HIGH', '5m': 'HIGH', '15m': 'HIGH' },
    trendDuration: 8,
    indicators: { ma5: 111400, ma20: 111350, rsi: 65 },
    reason: '強い上昇トレンド | すべての時間足でHIGH...'
  }
}
```

## 🎯 使い方

### 基本操作

1. **TheOptionにアクセス**: https://jp.theoption.com/trading
2. **ヘッダー確認**: 左上に「📊 データ収集中」ボタンが表示
3. **待機**: 約3分でシグナル生成開始（15分足の最小データ）
4. **確認**: 3回連続で同じシグナル → 確定表示
5. **詳細**: ボタンをクリックで詳細パネル展開

### シグナルの見方

#### 📊 データ収集中 (0-2/3)
- **意味**: まだ十分なデータが揃っていない
- **行動**: エントリーせず、データ蓄積を待つ

#### ⬆ HIGH (緑色)
- **意味**: 価格上昇の可能性が高い
- **推奨**: HIGHでエントリー
- **条件**: 3回連続でHIGH確認済み

#### ⬇ LOW (赤色)
- **意味**: 価格下降の可能性が高い
- **推奨**: LOWでエントリー
- **条件**: 3回連続でLOW確認済み

#### NEUTRAL (グレー)
- **意味**: トレンド不明瞭 or ボラティリティ不足
- **推奨**: エントリー見送り

### 信頼度の目安

- **80-90%**: 非常に強いシグナル（全時間足一致+長期継続）
- **70-80%**: 強いシグナル（複数時間足一致）
- **60-70%**: 中程度のシグナル
- **60%未満**: 様子見推奨

## ⚙️ 設定とカスタマイズ

### データ収集の初回待機時間

最低限必要なデータ:
- **1分足**: 50本 (50秒)
- **5分足**: 20本 (100秒 = 1分40秒)
- **15分足**: 20本 (300秒 = 5分)

→ **初回は約3-5分の待機が必要**

### 今後の改善予定 (v3.1)
- localStorage活用で2回目以降は即座に分析開始
- 過去データの保存・復元機能

## ⚠️ 注意事項

### 制限事項
- **初回待機時間**: 3-5分必要（15分足の最小データ）
- **ページリロード**: データがリセットされる（v3.1で改善予定）
- **Lightstreamer未統合**: DOMから価格取得（1秒間隔）

### 推奨事項
- 信頼度70%以上のシグナルを参考
- すべての時間足が一致しているシグナルを優先
- トレンド継続時間が長い（5分以上）シグナルが有利
- 他の分析手法と組み合わせて使用
- デモ口座で十分練習してから実運用

### 既知の課題
- localStorage未実装のため2回目以降も待機時間発生
- Lightstreamer直接統合未実装（より正確な価格取得）
